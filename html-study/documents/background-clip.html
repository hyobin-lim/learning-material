CSS background-clip 속성 이해하기

background-clip은 요소의 배경(색상/이미지)이 어디까지 채워질지를 결정하는 CSS 속성입니다.

값 설명 코드 예제 데모
border-box 배경이 테두리(border)까지 채워짐 (기본값) 
background-clip: border-box;
내용
padding-box 배경이 패딩(padding) 영역까지만 채워짐 
background-clip: padding-box;
내용
content-box 배경이 콘텐츠(content) 영역까지만 표시됨 
background-clip: content-box;
내용
text 배경이 텍스트 글자 모양에 맞게 잘려서 표시됨 
background-clip: text;
color: transparent;
