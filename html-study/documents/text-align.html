텍스트 정렬 및 간격 속성 정리

속성 설명 예제 코드 데모
line-height 줄 간격을 조절합니다. 
p { line-height: 3; }

이 문장은
줄 간격이
넓게 표시됩니다.
text-align: left 텍스트를 왼쪽 정렬합니다. 
p { text-align: left; }

왼쪽 정렬된 텍스트
text-align: center 텍스트를 가운데 정렬합니다. 
p { text-align: center; }

가운데 정렬된 텍스트
text-align: right 텍스트를 오른쪽 정렬합니다. 
p { text-align: right; }

오른쪽 정렬된 텍스트
text-align: justify 문단을 양쪽 끝에 맞춰 정렬합니다. 
p { text-align: justify; }

이 문장은 양쪽 끝에 맞춰 정렬됩니다. 긴 문장을 넣으면 양쪽이 가지런하게 보입니다.
text-align: start 글쓰기 방향에 따라 시작 부분에 맞춰 정렬합니다. (LTR에서는 왼쪽, RTL에서는 오른쪽) 
p { text-align: start; }

start 정렬된 텍스트 (왼쪽)
text-align: end 글쓰기 방향에 따라 끝 부분에 맞춰 정렬합니다. (LTR에서는 오른쪽, RTL에서는 왼쪽) 
p { text-align: end; }

end 정렬된 텍스트 (오른쪽)
text-indent 문단의 첫 줄을 들여쓰기 합니다. 
p { text-indent: 2em; }

이 문장은 첫 줄이 들여쓰기 되어 있습니다.
letter-spacing 글자 간격을 조절합니다. 
p { letter-spacing: 5px; }

글자 간격이 넓어진 텍스트
word-spacing 단어 간격을 조절합니다. 
p { word-spacing: 20px; }

단어 간격이 넓어진 텍스트 예시
vertical-align 인라인 요소나 테이블 셀에서 세로 정렬을 지정합니다. 
td { vertical-align: middle; }
위쪽 가운데 아래쪽
flex + align-items flex 컨테이너에서 자식 요소를 세로 방향으로 정렬합니다. 
.container { display:flex; align-items:center; }
