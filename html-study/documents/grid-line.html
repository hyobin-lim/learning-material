CSS Grid: 행과 열을 나누는 Grid Line 개념과 데모

그리드를 라인(경계선)으로 이해하고, grid-row / grid-column으로 정확히 배치하는 법을 코드와 함께 보여줍니다.

그리드를 라인으로 이해하기

  • 개념: CSS Grid는 칸 그 자체를 지정하는 게 아니라, 칸을 나누는 경계선인 grid line(라인) 번호를 기준으로 요소의 시작과 끝을 정의합니다.
  • 라인 수: N행이면 가로 라인은 N+1개, M열이면 세로 라인은 M+1개가 생깁니다.
  • 배치 원리: grid-row: a / b는 가로 라인의 a에서 시작해 b에서 끝나므로, 차지하는 행의 수는 b - a입니다. grid-column도 동일한 원리입니다.
세로 라인 (column) 번호 1 2 3 4 ┌─────┬─────┬─────┐ → 가로 라인 (row) 1 │ │ │ │ ├─────┼─────┼─────┤ → 가로 라인 (row) 2 │ │ │ │ ├─────┼─────┼─────┤ → 가로 라인 (row) 3 │ │ │ │ ├─────┼─────┼─────┤ → 가로 라인 (row) 4 │ │ │ │ └─────┴─────┴─────┘ → 가로 라인 (row) 5 ↓ ↓ ↓ ↓ 세로 라인은 열+1개, 가로 라인은 행+1개
  • 예시 해석: 4행×3열 그리드라면 가로 라인은 5개, 세로 라인은 4개입니다.
  • 라인 지정 예: grid-row: 2 / 5 → 가로 라인 2~4 사이(행 2~4)를 차지합니다.
  • 스팬 지정: grid-row: 1 / span 2처럼 span을 쓰면 “시작 라인부터 2칸 차지”라는 의미입니다.

예제와 코드, 데모를 한 표에 보기

설명 코드 데모
  • 그리드 구성: 3행×3열 그리드입니다.
  • item1: grid-row: 1 / 2, grid-column: 1 / 2 → 1행 1열의 단일 셀을 차지합니다.
  • item2: grid-row: 1 / 3, grid-column: 2 / 3 → 1~2행, 2열을 세로로 2칸 차지합니다.
  • item3: grid-row: 2 / 4, grid-column: 3 / 4 → 2~3행, 3열을 세로로 2칸 차지합니다.
  • 핵심: b - a가 차지하는 칸 수입니다. 예: 1 / 3 → 2칸.
 
.grid-demo-3x3 {
  display: grid;
  grid-template-columns: repeat(3, 88px);
  grid-template-rows: repeat(3, 88px);
  gap: 6px;
}

.item1 { grid-row: 1 / 2; grid-column: 1 / 2; }
.item2 { grid-row: 1 / 3; grid-column: 2 / 3; }
.item3 { grid-row: 2 / 4; grid-column: 3 / 4; }

  • 그리드 구성: 4행×4열 그리드입니다.
  • a: grid-column: 1 / 3, grid-row: 1 / 2 → 1행에서 1~2열(가로로 2칸) 차지.
  • b: grid-column: 3 / 5, grid-row: 1 / 3 → 1~2행, 3~4열(2×2 영역) 차지.
  • c: grid-column: 1 / 2, grid-row: 2 / 5 → 2~4행, 1열(세로로 3칸) 차지.
  • d: grid-column: 2 / 4, grid-row: 3 / 5 → 3~4행, 2~3열(2×2 영역) 차지.
  • 라인 번호: 4열이므로 세로 라인은 1~5, 4행이므로 가로 라인은 1~5입니다.
 
.grid-demo-4x4 {
  display: grid;
  grid-template-columns: repeat(4, 70px);
  grid-template-rows: repeat(4, 70px);
  gap: 6px;
}

.a { grid-column: 1 / 3; grid-row: 1 / 2; }
.b { grid-column: 3 / 5; grid-row: 1 / 3; }
.c { grid-column: 1 / 2; grid-row: 2 / 5; }
.d { grid-column: 2 / 4; grid-row: 3 / 5; }

라인 기반 배치 팁

  • span 문법: grid-column: 2 / span 3은 “세로 라인 2에서 시작해 3칸(열) 차지”를 의미합니다.
  • 음수 라인: -1은 마지막 라인을 의미합니다. 예: grid-column: 1 / -1 → 첫 라인부터 마지막 라인 전까지 전 열을 가로지름.
  • 축약형: grid-row: 2 / span 2 → 라인 2에서 시작하여 행 2칸 차지. grid-column도 동일.
  • 겹침 규칙: 같은 영역을 두 아이템이 요청하면 소스 순서에 따라 나중 요소가 위에 겹칩니다(명시적 z-index가 없을 때).
이 문서는 CSS Grid의 라인 개념을 중심으로 grid-row / grid-column 배치를 설명합니다.
