CSS 단축 속성(Shorthand Properties) 완전 정리

1. 단축 속성이란?

여러 개의 개별 속성을 한 줄로 묶어 간결하게 작성할 수 있는 방법입니다. 예를 들어 margin-top, margin-right, margin-bottom, margin-left를 각각 쓰는 대신 margin 단축 속성으로 한 번에 지정할 수 있습니다.

2. 주요 단축 속성 정리

속성적용 순서한글 설명세부 값 설명예제
margin 상 → 우 → 하 → 좌 바깥 여백 px, %, em 등 단위 사용 가능 margin: 10px 20px 30px 40px;
padding 상 → 우 → 하 → 좌 안쪽 여백 px, %, em 등 단위 사용 가능 padding: 5px 10px 15px 20px;
border 두께 → 스타일 → 색상 테두리 두께: thin, medium, thick, px 단위
스타일: solid, dashed, dotted, double, groove, ridge, inset, outset, none
색상: color 이름(red, blue 등), HEX(#ff0000), RGB(rgba(255,0,0,0.5)) 등 		border: 2px solid red;
background 색상 → 이미지 → 반복 → 위치 → 크기 배경 설정 색상: color 값
이미지: url("image.png")
반복: repeat, no-repeat, repeat-x, repeat-y
위치: left, right, center, top, bottom, px, %
크기: cover, contain, px, % 		background: #eee url(bg.png) no-repeat center/cover;
font 스타일 → 두께 → 크기/줄높이 → 글꼴 글꼴 관련 속성 스타일: normal, italic, oblique
두께: normal, bold, bolder, lighter, 100~900
크기: px, em, rem
줄높이: normal, 숫자, 단위
글꼴: "Arial", "Times New Roman", sans-serif 등 		font: italic bold 16px/1.5 "Arial", sans-serif;
list-style 형식 → 위치 → 이미지 목록 불릿 스타일 형식: disc, circle, square, decimal, lower-alpha 등
위치: inside, outside
이미지: url("icon.png") 		list-style: square inside url(icon.png);
flex 성장비율 → 축소비율 → 기준크기 Flexbox 아이템 크기 제어 성장비율: 남은 공간을 차지하는 비율
축소비율: 공간 부족 시 줄어드는 비율
기준크기: px, %, auto 		flex: 1 0 200px;
animation 이름 → 지속시간 → 타이밍 → 지연 → 반복횟수 → 방향 → 채우기모드 → 재생상태 애니메이션 설정 타이밍: ease, linear, ease-in, ease-out, ease-in-out
반복횟수: 1, infinite
방향: normal, reverse, alternate
채우기모드: none, forwards, backwards, both
재생상태: running, paused 		animation: bounce 2s ease-in-out 0s infinite alternate;
transition 속성 → 지속시간 → 타이밍 → 지연 변화 효과 속성: all, width, height, background 등
지속시간: s, ms
타이밍: ease, linear, ease-in, ease-out
지연: s, ms 		transition: all 0.5s ease-in-out;
grid 템플릿 → 행/열 간격 → 영역 Grid 레이아웃 템플릿: grid-template-rows / grid-template-columns
간격: gap, row-gap, column-gap
영역: grid-template-areas, grid-auto-flow 		grid: auto-flow / 200px 1fr;

3. 단축 속성 활용 예제와 데모

단축속성

위 박스는 margin, border, background, font, transition 단축 속성을 활용한 예제입니다. 마우스를 올리면 확대와 회전 효과가 적용됩니다.

4. 핵심 요약

  • 단축 속성은 여러 개별 속성을 한 줄로 묶어 작성할 수 있어 코드가 간결해집니다.
  • 순서가 중요한 속성(예: font, border, animation)은 반드시 올바른 순서로 작성해야 합니다.
  • 각 항목에 들어갈 수 있는 값들을 정확히 알고 있어야 에러 없이 사용할 수 있습니다.
  • 실제 프로젝트에서는 단축 속성을 적극 활용하면 가독성과 유지보수성이 크게 향상됩니다.
