CSS flex 속성 전체 정리

flex-grow (늘어남)

설명: 컨테이너에 남는 공간이 있을 때 아이템이 얼마나 늘어날지를 결정합니다.

.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }
.item3 { flex-grow: 3; }
grow=1
grow=2
grow=3

→ 남는 공간을 1:2:3 비율로 나눠 차지. 초록 박스가 가장 넓음.

flex-shrink (줄어듦)

설명: 컨테이너 공간이 부족할 때 아이템이 얼마나 줄어들지를 결정합니다.

.item1 { flex-shrink: 1; }
.item2 { flex-shrink: 2; }
.item3 { flex-shrink: 0; }
shrink=1
shrink=2
shrink=0

→ 화면을 줄이면 파란 박스가 두 배로 더 많이 줄어들고, 초록 박스는 줄어들지 않음.

flex-basis (기본 크기)

설명: 아이템의 기본 크기를 지정합니다.

.item1 { flex-basis: 50px; }
.item2 { flex-basis: 100px; }
.item3 { flex-basis: 200px; }
basis=50px
basis=100px
basis=200px

→ 기본 크기를 각각 50px, 100px, 200px으로 시작. 이후 grow/shrink 값에 따라 늘어나거나 줄어듦.

CSS flex 속성 전체 정리

전체 설명:
flex는 flex-grow, flex-shrink, flex-basis의 단축 속성입니다.
- flex-grow: 남는 공간을 얼마나 차지할지
- flex-shrink: 공간이 부족할 때 얼마나 줄어들지
- flex-basis: 기본 크기를 얼마로 할지

값에 따라 아이템의 크기와 공간 분배 방식이 달라집니다.
