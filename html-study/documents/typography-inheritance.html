타이포그래피 상속 (typography-inheritance)

속성 코드 예제 데모 설명
color .color-demo { color: darkred; }
이 텍스트는 darkred로 표시됩니다.
 텍스트 색상을 지정합니다. 상속되어 자식 요소에도 동일하게 적용되며, 전체 문서의 톤을 일관되게 유지하는 데 중요합니다.
font-family .family-demo { font-family: "Courier New", monospace; }
이 텍스트는 Courier New 글꼴로 표시됩니다.
 글꼴을 지정합니다. 상속되어 후손 텍스트에도 적용되어 일관된 폰트 체계를 유지하며, 문서 전체의 가독성과 분위기를 결정하는 핵심 속성입니다.
font-size .size-demo { font-size: 20px; }
이 텍스트는 20px 크기로 표시됩니다.
 글자 크기를 지정합니다. 상속되며, 상대 단위(em, %)를 사용하면 부모 크기에 따라 변합니다. 계층적 구조에서 크기 조절이 자연스럽게 이어집니다.
font-weight .weight-demo { font-weight: bold; }
이 텍스트는 굵게 표시됩니다.
 글자의 굵기를 지정합니다. 상속되어 자식 텍스트에도 영향을 주며, 강조나 구분을 위해 자주 사용됩니다.
font-style .style-demo { font-style: italic; }
이 텍스트는 이탤릭체로 표시됩니다.
 글자 스타일을 지정합니다. 상속되어 후손에도 적용되며, 인용문이나 강조 표현에 활용됩니다.
font-variant .variant-demo { font-variant: small-caps; }
이 텍스트는 스몰캡으로 표시됩니다.
 글자 변형을 지정합니다. 상속되어 후손에도 적용되며, 제목이나 특정 강조 구간에서 시각적 변화를 줄 수 있습니다.
text-shadow .shadow-demo { text-shadow: 2px 2px 4px gray; }
이 텍스트는 그림자가 적용됩니다.
 텍스트에 그림자를 추가합니다. 상속되어 자식 텍스트에도 동일하게 보이며, 입체감이나 강조 효과를 줄 때 사용됩니다.
