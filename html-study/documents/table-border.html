CSS table border 속성 정리

border는 표와 셀의 테두리를 지정하는 속성입니다.

실선 테두리 (solid)
과일가격

이중선 테두리 (double)
과일가격

속성 설명 예시 코드
border-width 테두리 두께 지정 border-width: 2px;
border-style 테두리 스타일 지정 (solid, dotted, dashed, double 등) border-style: dotted;
border-color 테두리 색상 지정 border-color: red;
border 단축 속성 (두께, 스타일, 색상 한 줄로) border: 2px solid black;

정리: border 속성은 표와 셀의 테두리를 지정하며, border-collapse와 함께 쓰면 겹치는 선을 하나로 합칠 수 있습니다.


✨ 핵심 포인트

border는 두께 + 스타일 + 색상을 한 줄로 지정 가능

border-collapse: collapse; → 셀 테두리 겹침 제거

다양한 스타일(solid, dotted, dashed, double)로 꾸밀 수 있음
