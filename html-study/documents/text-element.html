HTML 텍스트 관련 태그 정리

자주 쓰이는 텍스트 태그와 간단한 설명 및 예시입니다.

<p> — 문단 (Paragraph)

문단을 나타내는 블록 레벨 요소입니다. 문단마다 <p>를 사용합니다.

<p>이것은 문단입니다.</p>

<br> — 줄바꿈 (Line Break)

새 줄로 이동할 때 사용합니다. 자체 종료 태그로 블록을 만들지 않습니다.

첫 번째 줄<br>두 번째 줄

<h1> ~ <h6> — 제목 (Headings)

문서 구조를 위한 제목 태그. 숫자가 작을수록 더 중요한 제목입니다.

<h1>최상위 제목</h1>
<h2>섹션 제목</h2>

<strong> / <b> — 강조 (bold)

<strong>은 의미적 강조(중요함)를, <b>는 단순 시각적 굵게 처리에 사용합니다.

<strong>중요</strong>와 <b>굵게</b>

<em> / <i> — 기울임 (emphasis)

<em>은 의미적 강조(강조된 텍스트), <i>는 단순 기울임입니다.

<em>강조된 텍스트</em>와 <i>기울임</i>

<span> — 인라인 컨테이너

인라인 요소를 그룹화할 때 사용. 스타일이나 스크립트 적용 용도로 자주 사용됩니다.

<p>이 문장에 <span style="color:red">빨간 글자</span>가 있습니다.</p>

<div> — 블록 컨테이너

블록 레벨 컨테이너로 레이아웃 구획을 만들 때 사용합니다.

<div class="box">내용</div>

<ul> / <ol> / <li> — 목록

순서 없는 목록(<ul>)과 순서 있는 목록(<ol>)을 만듭니다.

<ol> 태그에는 type 속성을 지정할 수 있습니다.
- type="1" → 기본 숫자 (1,2,3…)
- type="A" → 대문자 알파벳 (A,B,C…)
- type="a" → 소문자 알파벳 (a,b,c…)
- type="I" → 로마 숫자 대문자 (I, II, III…)
- type="i" → 로마 숫자 소문자 (i, ii, iii…) 

<ol type="a">
  <li>첫 번째</li>
  <li>두 번째</li>
  <li>세 번째</li>
</ol>

<a> — 링크 (Anchor)

하이퍼링크를 만들 때 사용합니다. href 속성으로 목적지 지정.

<a href="https://www.example.com">예시 사이트</a>

<code> / <pre> — 코드 표기

<code>은 인라인 코드, <pre>는 공백과 줄바꿈을 유지하는 블록 코드에 사용합니다.

<pre><code>console.log('안녕');</code></pre>

<blockquote> — 인용문

긴 인용문을 표시할 때 사용합니다. 출처를 함께 표기하면 좋습니다.

<blockquote>이것은 인용문입니다.</blockquote>

기타 유용한 태그들

  • <small>: 작은 글씨
  • <abbr>: 약어 (title 속성으로 전체 의미 제공)
  • <mark>: 하이라이트
  • <del> / <ins>: 삭제된/삽입된 텍스트
  • <sub> / <sup>: 아래첨자/위첨자
  • <time>: 날짜/시간 표기
<p>수정된 날짜: <time datetime="2025-11-21">2025-11-21</time></p>

간단한 정리였습니다. 필요한 태그를 골라 예제에니 적용해 보세요.
