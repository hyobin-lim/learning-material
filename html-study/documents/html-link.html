HTML 링크란 무엇인가?

HTML 링크(Anchor)는 웹 문서에서 다른 문서, 같은 문서 내 특정 위치, 외부 자원(이미지, 폰트, 미디어 등)으로 연결할 수 있는 가장 기본적인 요소입니다.
기본 문법은 <a href="URL">링크 텍스트</a> 형태이며, 웹의 핵심인 **하이퍼텍스트(Hypertext)**를 가능하게 합니다.

HTML 링크의 큰 틀

링크는 단순히 다른 페이지로 이동하는 것뿐만 아니라, 다양한 자원과 연결되어 웹을 풍부하게 만듭니다. 이번 5장에서는 링크를 다음과 같이 구분하여 설명합니다:

  • 링크 연결 — 기본적인 페이지 간 이동과 앵커 활용
  • 미디어 연결 — 이미지, 오디오, 비디오 등 외부 자원 연결
  • 폰트 연결 — 웹폰트 불러오기와 스타일 적용
  • 실무 활용 팁과 브라우저 지원 현황 — 실제 프로젝트에서의 활용과 호환성

정리

HTML 링크는 웹의 뼈대를 이루는 요소로, 단순한 이동에서부터 다양한 자원 연결까지 확장됩니다. 아래 문서들을 통해 각 영역을 상세히 살펴보면, 링크의 활용 범위를 전체적으로 이해할 수 있습니다.
