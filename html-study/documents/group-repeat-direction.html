반복·방향 그룹

반복·방향 그룹은 애니메이션의 반복 횟수와 재생 방향을 제어합니다. 같은 애니메이션이라도 반복 횟수와 방향 설정에 따라 전혀 다른 느낌을 줄 수 있습니다.

1. animation-iteration-count

애니메이션을 몇 번 반복할지 지정합니다. 정수(예: 1, 2, 3…) 또는 infinite. 기본값은 1.

값설명예제 코드데모
1 한 번만 실행 (hover 시) 
.demo.iter1:hover { animation: moveBox 2s 1; }
3 세 번 반복 (hover 시) 
.demo.iter3:hover { animation: moveBox 2s 3; }
infinite 무한 반복 
.box { animation: moveBox 2s infinite; }

2. animation-direction

애니메이션이 반복될 때 어떤 방향으로 재생될지 지정합니다.
@keyframes의 재생 방향. normal, reverse, alternate, alternate-reverse. 기본값은 normal.

값설명예제 코드데모
normal 항상 정방향으로 실행 
.box { animation: moveBox 2s infinite normal; }
reverse 항상 역방향으로 실행 
.box { animation: moveBox 2s infinite reverse; }
alternate 정방향 → 역방향 번갈아 실행 
.box { animation: moveBox 2s infinite alternate; }
alternate-reverse 역방향 → 정방향 번갈아 실행 
.box { animation: moveBox 2s infinite alternate-reverse; }

정리: 반복·방향 그룹은 애니메이션의 반복 횟수와 방향을 제어하여 다양한 변화를 줄 수 있습니다.
