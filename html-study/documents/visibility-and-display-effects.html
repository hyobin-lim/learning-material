표시/가시성으로 인한 간접적 파급 (visibility-and-display-effects)

속성 코드 예제 데모 설명
visibility .visibility-demo { visibility: hidden; }
숨겨진 텍스트 보이는 텍스트
요소를 숨기지만 공간은 유지합니다. 부모가 숨겨지면 자식도 함께 보이지 않아 후손까지 영향을 줍니다.
opacity .opacity-demo { opacity: 0.5; }
반투명하게 표시된 텍스트
 요소와 자식 전체가 투명해집니다. 투명도는 상속되진 않지만, 부모의 합성 결과가 후손에도 적용되어 전체 묶음이 흐려집니다.
transform .transform-demo { transform: rotate(10deg) scale(1.2); }
회전/확대된 박스
 요소를 회전·이동·확대/축소합니다. 좌표계가 바뀌어 자식 요소의 배치에도 간접적으로 영향을 줍니다.
display .display-none { display: none; }
숨겨진 요소 보이는 요소
요소를 완전히 제거해 공간도 사라집니다. 부모가 none이면 자식도 함께 렌더링되지 않습니다.
clip-path .clip-demo { clip-path: circle(40%); }
요소의 표시 영역을 잘라내어 특정 모양만 보이게 합니다. 자식 콘텐츠도 잘린 영역 안에서만 표시됩니다.
filter .filter-demo { filter: blur(2px); }
흐려진 텍스트
 요소와 자식 전체에 그래픽 효과(흐림, 색상 변경 등)를 적용합니다. 부모에 적용하면 후손도 함께 영향을 받습니다.
pointer-events .pointer-demo { pointer-events: none; }
클릭 불가능한 박스
 요소와 자식의 마우스 이벤트 반응을 제어합니다. none이면 클릭이나 hover가 무시되어 자식까지 영향을 줍니다.
