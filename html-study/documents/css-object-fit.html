CSS object-fit 속성 이해하기

object-fit 속성은 <img>나 <video> 요소가 고정된 크기의 컨테이너 안에서 어떻게 맞춰질지를 정의합니다. 이미지나 영상의 비율을 유지하거나, 잘라내거나, 늘려서 보여줄 때 유용합니다.

설명과 예제 코드

값 설명 예제 코드
fill 컨테이너를 꽉 채우도록 이미지가 늘어나거나 줄어듭니다. 비율은 유지되지 않습니다. 
img { object-fit: fill; }
contain 이미지의 비율을 유지하면서 컨테이너 안에 모두 들어오도록 조정합니다. 여백이 생길 수 있습니다. 
img { object-fit: contain; }
cover 이미지의 비율을 유지하면서 컨테이너를 가득 채웁니다. 일부 영역은 잘릴 수 있습니다. 
img { object-fit: cover; }
none 이미지를 원래 크기 그대로 표시합니다. 컨테이너보다 크면 넘칩니다. 
img { object-fit: none; }
scale-down none과 contain 중 더 작은 크기로 표시합니다. 
img { object-fit: scale-down; }
