CSS background-repeat 속성 이해하기

background-repeat은 요소의 배경 이미지가 어떻게 반복될지를 결정하는 CSS 속성입니다.

값 설명 코드 예제 데모
repeat 가로와 세로 모두 반복 (기본값) 
background-repeat: repeat;
repeat-x 가로 방향으로만 반복 
background-repeat: repeat-x;
repeat-y 세로 방향으로만 반복 
background-repeat: repeat-y;
no-repeat 반복하지 않고 한 번만 표시 
background-repeat: no-repeat;
round 이미지를 요소 크기에 맞게 늘려 반복 
background-repeat: round;
space 이미지를 반복하되 남는 공간을 균등하게 분배 
background-repeat: space;
