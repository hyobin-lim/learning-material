CSS 선택자 명령어 정리

CSS 선택자는 특정 HTML 요소를 선택하여 스타일을 적용할 때 사용합니다. 아래 표는 대표적인 선택자와 예시를 정리한 것입니다.

선택자 설명 예시
* 전체 선택자 (모든 요소 선택) * { margin:0; padding:0; }
태그명 특정 태그 선택 p { color:blue; }
.classname 클래스 선택자 .highlight { background:yellow; }
#idname ID 선택자 #main { width:960px; }
parent child
부모 자식		 하위(자손) 선택자 div p { font-size:14px; }
parent > child
부모 > 자식		 직계 자식 선택자 ul > li { list-style:none; }
element1 + element2 바로 뒤에 오는 형제 선택자 h1 + p { color:red; }
element1 ~ element2 뒤에 오는 모든 형제 선택자 h1 ~ p { color:green; }
[attribute] 속성 선택자 input[type="text"] { border:1px solid #ccc; }
태그:hover 가상 클래스 선택자 (마우스 오버) a:hover { color:orange; }
태그:nth-child(n) 특정 순서의 자식 선택 li:nth-child(2) { color:blue; }

정리

  • 전체 선택자: 모든 요소에 적용
  • 태그/클래스/ID 선택자: 특정 요소에 적용
  • 계층 선택자: 부모-자식, 형제 관계를 기반으로 선택
  • 속성 선택자: 특정 속성을 가진 요소 선택
  • 가상 클래스 선택자: 상태(hover, nth-child 등)에 따라 선택
