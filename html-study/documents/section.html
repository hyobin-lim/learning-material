<section> 태그 정리

문서의 주제를 가진 구획을 의미하며, 관련된 콘텐츠를 묶어 구조적으로 표현할 때 사용합니다.

쉬운 설명 코드 예제 데모 화면 베스트 프랙티스 / 주의사항

의미: <section> 태그는 문서 내에서 특정 주제를 가진 콘텐츠 묶음을 나타냅니다. 예를 들어 “소개”, “서비스”, “연락처”와 같은 구획을 나눌 때 사용합니다.

사용 위치: 페이지 본문에서 논리적으로 구분되는 영역을 나눌 때 사용합니다. 각 <section>은 보통 제목(<h2> 등)을 포함하여 그 주제를 명확히 합니다.

접근성: 스크린리더는 <section>을 인식하여 문서의 구조를 알려주므로, 반드시 제목을 포함하는 것이 좋습니다.

 
<section>
  <h2>서비스 소개</h2>
  <p>저희가 제공하는 주요 서비스 목록입니다.</p>
  <ul>
    <li>웹 개발</li>
    <li>디자인</li>
    <li>컨설팅</li>
  </ul>
</section>

서비스 소개

저희가 제공하는 주요 서비스 목록입니다.

  • 웹 개발
  • 디자인
  • 컨설팅
  • 주제 중심: 각 <section>은 반드시 주제를 가져야 합니다.
  • 제목 포함: <h2> 이상의 제목을 포함하여 구획의 의미를 명확히 하십시오.
  • 구조적 구분: 단순히 꾸미기 위해 쓰지 말고, 문서의 논리적 구조를 표현할 때 사용하십시오.
  • 중첩 가능: <section> 안에 또 다른 <section>을 넣을 수 있습니다.
