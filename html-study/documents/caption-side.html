CSS caption-side 속성 정리

caption-side는 표 제목(<caption>)을 위 또는 아래에 배치합니다.

위쪽 캡션 (caption-side: top)
과일가격
사과3000원
바나나2000원
아래쪽 캡션 (caption-side: bottom)
과일가격
포도4000원
오렌지3500원
값 설명 예시 코드
top 캡션을 표 위쪽에 표시 (기본값) caption-side: top;
bottom 캡션을 표 아래쪽에 표시 caption-side: bottom;
inherit 부모 요소의 값을 상속 caption-side: inherit;
initial 기본값(top)으로 초기화 caption-side: initial;

정리: caption-side는 표 제목을 위(top) 또는 아래(bottom)에 배치하는 속성으로, 기본값은 top입니다.
