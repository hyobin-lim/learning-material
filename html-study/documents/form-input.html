Form 섹션 2 - 입력 요소

전체 설명

입력 요소(Input Elements)는 사용자가 **직접 값을 입력할 수 있는 필드**를 말합니다.
대표적으로 <input> (인풋 태그)과 <textarea> (텍스트영역 태그)가 있으며,
텍스트, 비밀번호, 이메일, 전화번호, URL, 숫자, 날짜 등 다양한 타입을 통해 데이터를 입력받습니다.
즉, 입력 요소는 **사용자가 키보드로 직접 데이터를 입력하는 모든 필드**를 의미합니다.

상세 설명

<input> (인풋 태그)은 단일 줄 입력 필드이고, type 속성으로 입력 종류를 지정합니다.
<textarea> (텍스트영역 태그)는 여러 줄 입력이 가능합니다.
placeholder (플레이스홀더)는 입력 힌트를 보여주고,
maxlength/minlength (최대/최소 길이)는 입력 글자 수를 제한합니다.
readonly (읽기전용)은 값만 표시하고 수정은 불가,
disabled (비활성화)는 입력 불가이며 서버 전송에서도 제외됩니다.

자주 쓰는 속성들:
• placeholder (플레이스홀더): 입력칸에 힌트 텍스트 표시
• minlength / maxlength (최소/최대 글자 수): 입력 길이 제한
• readonly (읽기전용): 값은 보이지만 수정 불가
• disabled (비활성화): 입력 불가, 서버 전송도 제외
• autocomplete (자동완성): 브라우저 자동완성 켜기/끄기
• pattern (패턴): 정규식으로 입력 형식 제한
항목설명예제 코드데모
text (텍스트) 일반 텍스트 입력 
<input type="text" placeholder="이름" minlength="2" maxlength="30" required>
password (비밀번호) 비밀번호 입력, 입력값은 ●●●로 표시 
<input type="password" minlength="8" required>
email (이메일) 이메일 형식 검증 지원 
<input type="email" autocomplete="email" placeholder="you@example.com" required>
tel (전화번호) + pattern (패턴) 전화번호 형식 검증 
<input type="tel" pattern="^0[0-9]{1,2}-[0-9]{3,4}-[0-9]{4}$" placeholder="02-1234-5678">
url (주소) 웹 주소(URL) 입력 
<input type="url" placeholder="https://example.com">
textarea (텍스트영역) 여러 줄 텍스트 입력 
<textarea rows="4" cols="40" maxlength="300" placeholder="자기소개"></textarea>
readonly / disabled 읽기전용 vs 비활성화 
<input value="편집 불가" readonly>
<input value="비활성" disabled>
datalist (데이터목록) 자동완성 후보 목록 제공 
<input list="cities">
<datalist id="cities">
  <option value="Seoul">
  <option value="Busan">
</datalist>
