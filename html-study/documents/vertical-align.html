CSS 속성 완전 정리: vertical-align

1. 개념

vertical-align은 CSS에서 인라인 요소와 table-cell 요소의 콘텐츠를 세로 방향으로 정렬하는 속성입니다.
텍스트 줄(line box) 안에서 이미지·글자 위치를 맞추거나, 테이블 셀 안에서 내용을 위·중간·아래로 배치할 때 사용합니다.

2. 대표 값

값 설명 활용 예시
baseline 부모 요소의 기준선에 맞춤 (기본값) 텍스트와 이미지 하단 맞추기
top 줄 또는 셀의 위쪽에 맞춤 테이블 셀 상단 정렬
middle 줄 또는 셀의 중앙에 맞춤 세로 중앙 정렬
bottom 줄 또는 셀의 아래쪽에 맞춤 테이블 셀 하단 정렬
text-top / text-bottom 부모 텍스트의 위/아래에 맞춤 텍스트와 아이콘 정렬
sub / super 아래첨자/위첨자 위치 수학식, 화학식 표기

3. 예제 코드 + 설명 + 데모

.cell {
  display: table-cell;   /* 셀처럼 동작 */
  vertical-align: middle; /* 세로 중앙 정렬 */
  text-align: center;     /* 가로 중앙 정렬 */
}

위 코드는 display: table-cell로 지정된 요소 안에서 콘텐츠를 세로 중앙에 배치합니다. vertical-align은 블록 요소에는 적용되지 않고, 인라인 요소와 테이블 셀에서만 동작합니다.

위(top)
가운데(middle)
아래(bottom)

4. 인라인 요소 데모

인라인 요소에서도 vertical-align을 사용해 상대적 위치를 조정할 수 있습니다.

텍스트 기준선 text-top text-bottom super sub

5. 실무 활용 사례: 아이콘과 텍스트 정렬

아이콘과 텍스트를 나란히 배치할 때 vertical-align: middle을 사용하면 깔끔하게 정렬됩니다.

아이콘과 텍스트 정렬

6. 특징과 주의사항

  • 블록 요소에는 적용되지 않음 → 인라인/테이블 셀 전용
  • 인라인 요소의 세로 정렬은 line-height에 영향을 받음
  • 세로 중앙 정렬을 직관적으로 구현할 수 있음
  • 현대 레이아웃에서는 Flexbox/Grid로 대체 가능

7. 최신 대체 기술 비교

Flexbox: align-items: center;로 세로 중앙 정렬 가능
Grid: align-items, align-self로 세로 정렬 가능
→ 최신 레이아웃에서는 Flex/Grid를 기본으로 사용하고, vertical-align은 제한적인 상황에서만 활용

8. 핵심 요약

vertical-align은 인라인 요소와 테이블 셀의 세로 정렬을 제어하는 속성입니다.
대표값은 baseline, top, middle, bottom 등이며, 특히 세로 중앙 정렬을 간단히 구현할 수 있다는 장점이 있습니다.
하지만 블록 요소에는 적용되지 않으며, 최신 레이아웃에서는 Flexbox와 Grid가 더 자주 쓰입니다.
실무에서는 아이콘과 텍스트 정렬, 수학식/화학식 표기, 간단한 테이블 레이아웃에서 유용하게 활용됩니다.
