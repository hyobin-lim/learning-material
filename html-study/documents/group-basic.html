기본 제어 그룹

기본 제어 그룹은 애니메이션의 가장 핵심적인 흐름을 제어하는 속성들로 구성됩니다. 어떤 애니메이션을 사용할지, 얼마나 오래 실행할지, 어떤 속도로 진행할지, 언제 시작할지를 결정합니다.

포함 속성

1. animation-name

어떤 @keyframes를 사용할지 지정합니다. 값은 키프레임의 식별자이며, none은 애니메이션을 비활성화합니다.

값 설명 예제 코드 데모
moveBox 좌→우로 이동 
.box { animation-name: moveBox; animation-duration: 2s; }
fadeIn 투명 → 불투명 
.box { animation-name: fadeIn; animation-duration: 2s; }
none 애니메이션 없음 
.box { animation-name: none; }
 —

2. animation-duration

애니메이션이 한 사이클을 실행하는 시간을 지정합니다. 단위는 s(초) 또는 ms(밀리초). 기본값은 0s(즉시 적용, 실제 애니메이션 없음)입니다.

값 설명 예제 코드 데모
3s 3초 동안 실행 
.box { animation: moveBox 3s infinite alternate; }
800ms 0.8초 동안 실행 
.box { animation: moveBox 800ms infinite alternate; }

3. animation-timing-function

시간에 따른 진행 속도 곡선을 지정합니다. 기본값은 ease. 대표 값: linear, ease, ease-in, ease-out, ease-in-out, cubic-bezier(x1,y1,x2,y2), steps(n, start|end).

값 설명 예제 코드 데모
linear 일정한 속도로 진행 
.box { animation: moveBox 2s linear infinite alternate; }
ease-in 천천히 시작 
.box { animation: moveBox 2s ease-in infinite alternate; }
ease-out 천천히 끝남 
.box { animation: moveBox 2s ease-out infinite alternate; }
ease-in-out 시작·끝 부드럽게 
.box { animation: scaleUp 2s ease-in-out infinite alternate; }
steps(4, end) 4단계로 끊어 진행 
.box { animation: moveBox 2s steps(4, end) infinite alternate; }

animation-timing-function: cubic-bezier()

cubic-bezier(x1, y1, x2, y2)는 네 개의 좌표값을 받아 속도 곡선을 직접 정의합니다. 값에 따라 애니메이션의 가속·감속·탄성 효과 등이 달라집니다.

값 설명 예제 코드 데모
cubic-bezier(0.25, 0.1, 0.25, 1) 기본 ease와 유사 (천천히 시작·끝) 
.box { animation: moveBox 2s cubic-bezier(0.25,0.1,0.25,1) infinite alternate; }
cubic-bezier(1,0,1,1) ease-in (빠르게 가속) 
.box { animation: moveBox 2s ccubic-bezier(1,0,1,1) infinite alternate; }
cubic-bezier(0,0,0.58,1) ease-out (빠르게 감속) 
.box { animation: moveBox 2s cubic-bezier(0,0,0.58,1) infinite alternate; }
cubic-bezier(0.68, -0.55, 0.27, 1.55) 탄성 효과 (overshoot) 
.box { animation: moveBox 2s cubic-bezier(0.68,-0.55,0.27,1.55) infinite alternate; }
cubic-bezier(0.4, 0, 0.2, 1) Material Design 표준 곡선 
.box { animation: moveBox 2s cubic-bezier(0.4,0,0.2,1) infinite alternate; }

4. animation-delay

애니메이션 시작 지연 시간을 지정합니다. 음수 지연은 해당 경과 시점에서 즉시 시작합니다. 단위는 s 또는 ms.

값 설명 예제 코드 데모
1s 1초 뒤 시작 
.box { animation: colorChange 2s ease infinite alternate; animation-delay: 1s; }
-0.5s 0.5초 진행된 상태로 시작 
.box { animation: colorChange 2s ease infinite alternate; animation-delay: -0.5s; }

정리: 기본 제어 그룹은 애니메이션의 실행 흐름을 제어하는 가장 중요한 속성들입니다.
