CSS 레이아웃 완전 정리: display: table / table-cell

1. 개념

display: table과 display: table-cell은 HTML의 <table> 태그처럼 요소를 행과 열 구조로 배치할 수 있게 해주는 CSS 속성입니다. 부모 요소를 table로 지정하면 내부 자식 요소들을 table-row, table-cell로 배치할 수 있습니다.

2. 대표 속성 및 값

속성대표값설명
displaytable, table-row, table-cell, table-caption요소를 테이블 구조처럼 동작시킴
vertical-aligntop, middle, bottom셀 내부 콘텐츠의 세로 정렬
border-collapsecollapse, separate테두리 병합 여부 제어 (원래 table 태그에서 사용)

3. 예제 코드 + 설명 + 데모

.table-demo {
  display: table;        /* 부모 요소를 테이블처럼 동작시킴 */
  border: 1px solid #333;
  width: 100%;
}
.row {
  display: table-row;    /* 자식 요소를 행(row)처럼 동작시킴 */
}
.cell {
  display: table-cell;   /* 자식 요소를 셀(cell)처럼 동작시킴 */
  border: 1px solid #999;
  padding: 10px;
  vertical-align: middle; /* 셀 내부 콘텐츠를 세로 중앙 정렬 */
  text-align: center;     /* 셀 내부 콘텐츠를 가로 중앙 정렬 */
}

위 코드는 부모 요소를 display: table로 지정해 테이블처럼 동작하게 하고, 자식 요소를 table-row, table-cell로 지정해 행과 셀 구조를 만듭니다. 이렇게 하면 HTML의 <table> 태그 없이도 테이블 레이아웃을 구현할 수 있습니다.

4. Flexbox와 비교

Flexbox는 1차원 레이아웃(가로 또는 세로 한 방향)에 강력하며, 반응형 레이아웃에 적합합니다. 반면 table-cell은 2차원 행·열 구조를 흉내내지만 유연성이 떨어집니다.

5. 장점과 단점

  • 장점: HTML <table> 없이도 테이블 레이아웃 구현 가능, 셀 높이가 자동으로 맞춰짐, vertical-align 사용 가능
  • 단점: Flexbox/Grid에 비해 유연성이 떨어지고 반응형 레이아웃에는 적합하지 않음
  • 현재 활용: 과거에는 많이 쓰였지만 지금은 Flex/Grid가 대체, 보조적으로만 사용

5-1. 장점 상세 설명

① HTML <table> 없이도 테이블 레이아웃 구현 가능

원래 행·열 구조는 <table>, <tr>, <td> 태그로만 만들 수 있었지만, CSS의 display: table, display: table-row, display: table-cell을 사용하면 일반적인 <div> 같은 블록 요소도 테이블처럼 동작합니다. 덕분에 HTML 구조를 더 의미 있게 유지하면서도 테이블 레이아웃을 흉내낼 수 있습니다.

② 셀 높이가 자동으로 맞춰짐

테이블 레이아웃의 특징은 같은 행에 있는 셀들의 높이가 자동으로 동일하게 맞춰진다는 점입니다. 한 셀에 긴 텍스트가 들어가면 그 행의 다른 셀들도 같은 높이를 가지게 되어 균형 잡힌 레이아웃을 쉽게 만들 수 있습니다.

③ vertical-align 사용 가능

일반 블록 요소에서는 vertical-align(세로 정렬) 속성이 잘 적용되지 않지만, table-cell에서는 세로 정렬이 가능합니다. 대표값은 top, middle, bottom이며, 특히 세로 중앙 정렬을 직관적으로 구현할 수 있습니다.

6. 실무 팁

  • 간단한 행·열 구조를 빠르게 만들 때 유용
  • 세로 중앙 정렬이 필요할 때 vertical-align: middle을 활용
  • 반응형 웹에서는 Flex/Grid를 기본으로 사용하고, table-cell은 제한적인 상황에서만 사용
  • 이메일 템플릿이나 레거시 브라우저 호환성 작업에서 여전히 쓰임

7. 핵심 요약

display: table / table-cell은 CSS로 테이블 구조를 흉내내는 레이아웃 방식입니다.셀 높이 자동 맞춤과 세로 정렬이 장점이지만, 현대 웹에서는 Flexbox와 Grid가 더 강력하고 유연하기 때문에 보조적으로만 사용됩니다.
