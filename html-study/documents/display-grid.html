CSS Grid 완전 정리

1. CSS Grid란?

CSS Grid는 웹 레이아웃을 위한 2차원 시스템으로, 행(row)과 열(column)을 동시에 제어할 수 있습니다. Flexbox가 1차원 레이아웃이라면 Grid는 2차원 레이아웃을 지원한다는 점이 가장 큰 차이입니다.

2. 기본 개념

  • Grid Container: display: grid를 선언한 부모 요소
  • Grid Item: Grid Container의 직계 자식 요소
  • Grid Line: 행과 열을 구분하는 선
  • Grid Track: 두 Grid Line 사이의 공간 (행 또는 열)
  • Grid Cell: 네 개의 Grid Line으로 둘러싸인 최소 단위
  • Grid Area: 여러 셀을 합친 직사각형 영역

3. 주요 속성과 값

속성설명가능한 값예제
display 컨테이너를 Grid로 선언 grid, inline-grid .container { display: grid; }
grid-template-columns 열의 개수와 크기 정의 px, %, fr, auto, repeat(), minmax() grid-template-columns: 200px 1fr 2fr;
grid-template-rows 행의 개수와 크기 정의 px, %, fr, auto, repeat(), minmax() grid-template-rows: 100px auto;
gap 셀 간격 설정 px, em, rem (행과 열 모두 적용) gap: 20px;
row-gap / column-gap 행 또는 열 간격만 설정 px, em, rem row-gap: 10px; column-gap: 20px;
grid-column / grid-row 아이템의 위치 지정 시작 라인 번호 / 끝 라인 번호 .item { grid-column: 1 / 3; }
grid-template-areas 영역 이름으로 레이아웃 정의 문자열로 영역 이름 지정 grid-template-areas: "header header" "sidebar main";
justify-items 아이템의 수평 정렬 start, end, center, stretch justify-items: center;
align-items 아이템의 수직 정렬 start, end, center, stretch align-items: start;
place-items 수평·수직 정렬 단축 속성 위 두 속성을 함께 지정 place-items: center;
justify-content 전체 그리드의 수평 정렬 start, end, center, stretch, space-around, space-between, space-evenly justify-content: space-between;
align-content 전체 그리드의 수직 정렬 start, end, center, stretch, space-around, space-between, space-evenly align-content: center;
place-content 수평·수직 정렬 단축 속성 위 두 속성을 함께 지정 place-content: center space-between;

6. 고급 기능

  • repeat(): 반복되는 패턴을 간단히 정의 (repeat(3, 1fr))
  • minmax(): 최소/최대 크기 지정 (minmax(100px, auto))
  • fr 단위: 남은 공간을 비율로 나눔 (1fr, 2fr)
  • subgrid: 자식 그리드가 부모 그리드 라인을 공유 (최신 기능)
  • masonry: 벽돌식 레이아웃 지원 (실험적 기능)

7. 핵심 요약

  • CSS Grid = 2차원 레이아웃 시스템 (행+열 동시 제어)
  • Flexbox와 차이점: Flexbox는 1차원, Grid는 2차원
  • 주요 속성: grid-template-columns, grid-template-rows, gap, grid-area, justify-content, align-content
  • 값 설명: px, %, fr, auto, repeat(), minmax() 등 다양한 단위와 함수 활용 가능
  • 실전 활용: 반응형 레이아웃, 복잡한 페이지 구조를 간단히 구현 가능

8. 추가 팁

  • 모바일 퍼스트: 작은 화면부터 레이아웃을 정의하고 점차 확장
  • Grid + Media Query: 화면 크기에 따라 grid-template-columns를 변경하면 반응형 레이아웃 구현 가능
  • Flexbox와 혼합 사용: 큰 틀은 Grid로, 세부 정렬은 Flexbox로 처리하면 효율적
  • 브라우저 지원: 대부분의 최신 브라우저에서 지원, 단 subgrid는 일부만 지원
