HTML의 기본 문법

HTML 문서는 웹 페이지의 뼈대 역할을 하며, 계층 구조를 갖는 태그들로 구성되어 있습니다.

태그의 사용법

태그를 표시하는 기호는 엔티티 코드로 '&lt;' 와 '&gt;' 로 표현합니다. 괄호 안에 명령어(태그 이름)를 넣어 사용합니다.

태그는 여는 태그와 닫는 태그로 구성됩니다.
여는 태그: <태그이름>
닫는 태그: </태그이름>
예시: <p>문단 내용</p>

일부 태그는 닫는 태그가 필요 없는 경우도 있습니다. 예: <img>, <br>, <hr>

공백과 특수문자

HTML은 여러 개의 공백을 하나로 처리하기 때문에, 의도적으로 공백을 표시하려면 &nbsp; (non-breaking space)를 사용합니다.

예: 사과&nbsp;&nbsp;바나나 → 출력: 사과  바나나

그 외에도 다양한 엔티티 코드가 있습니다. 예: &amp; → &, &quot; → "

HTML 문서의 기본 구조

HTML 문서는 다음과 같은 기본 구조를 가집니다:

<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <title>문서 제목</title>
  </head>
  <body>
    <h1>제목</h1>
    <p>본문 내용</p>
  </body>
</html>
