CSS 미디어쿼리(Media Query) 이해하기

미디어쿼리(Media Query)는 화면 크기, 해상도, 방향 등 조건에 따라 다른 스타일을 적용할 수 있게 해주는 CSS 기능입니다.
반응형 웹 디자인의 핵심으로, PC·태블릿·모바일 환경에 맞는 UI를 제공합니다.

1. 예제와 데모

/* 데모용 미디어쿼리 */
@media (max-width: 600px) {
  .demo-box {
    background: #ff7675;
    color: #fff;
    transform: rotate(45deg);
  }
}

@media (min-width: 601px) and (max-width: 1024px) {
  .demo-box {
    background: #74b9ff;
    color: #fff;
    transform: scale(1.2);
  }
}

@media (min-width: 1025px) {
  .demo-box {
    background: #55efc4;
    color: #111;
  }
}

@media (orientation: landscape) {
  .demo-box {
    border-radius: 50%;
  }
}

정리 표

구분 조건 스타일 변화
모바일 max-width: 600px 배경 빨강(#ff7675), 글자 흰색, transform: rotate(45deg)
태블릿 601px ~ 1024px 배경 파랑(#74b9ff), 글자 흰색, transform: scale(1.2)
데스크톱 min-width: 1025px 배경 초록(#55efc4), 글자 검정
가로 방향 orientation: landscape border-radius: 50% → 박스가 원형으로 표시됨

2. 기본 문법 구조

형식 설명 예제
@media (조건) { CSS } 조건이 참일 때 CSS 적용 @media (max-width: 600px) { body { background: yellow; } }
and 조건 결합 @media (min-width: 600px) and (orientation: landscape)
not 조건 부정 @media not all and (monochrome)
only 특정 미디어타입만 적용 @media only screen and (max-width: 768px)

3. 주요 조건 요소

요소 설명 예제
max-width 지정한 너비 이하 @media (max-width: 600px)
min-width 지정한 너비 이상 @media (min-width: 1024px)
orientation 화면 방향 @media (orientation: landscape)
resolution 화면 해상도 @media (min-resolution: 2dppx)
aspect-ratio 화면 비율 @media (aspect-ratio: 16/9)
prefers-color-scheme 다크/라이트 모드 @media (prefers-color-scheme: dark)

4. 활용 팁

  • 모바일 퍼스트: 작은 화면부터 스타일 적용 후 확장
  • 조건 결합: and, not, only 활용
  • 테스트: 브라우저 개발자 도구로 다양한 화면 크기 확인
  • 실전 패턴: min-width 중심으로 작성하면 유지보수에 유리
