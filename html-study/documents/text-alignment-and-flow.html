텍스트 정렬과 흐름 (text-alignment-and-flow)

속성 코드 예제 데모 설명
text-align .example1 { text-align: center; }
가운데 정렬된 텍스트
 블록 컨테이너 내부의 인라인 콘텐츠를 좌·우·가운데·양쪽으로 정렬합니다. 자식 텍스트와 인라인 요소에도 영향을 주며, 블록 자체의 위치는 바꾸지 않습니다. 블록을 가운데 두려면 margin: 0 auto; 같은 별도 기법을 사용해야 합니다.
direction .example2 { direction: rtl; }
오른쪽에서 왼쪽으로 흐르는 텍스트
 텍스트의 기본 흐름 방향을 지정합니다. rtl은 줄 시작과 끝의 의미를 바꾸며 인라인 배치와 줄바꿈 규칙이 자식에게 상속됩니다. 아랍어·히브리어 같은 언어에 필수적입니다.
writing-mode .example3 { writing-mode: vertical-rl; }
세로쓰기 예제
 글쓰기 방향을 가로·세로로 바꿉니다. 줄 진행 방향과 인라인 축이 변경되어 자식 텍스트 레이아웃에도 연쇄적으로 적용됩니다. 일본어·중국어 세로쓰기에서 자주 사용됩니다.
text-indent .example4 { text-indent: 2em; }
첫 줄 들여쓰기 적용
 블록의 첫 줄을 들여쓰기합니다. 상속되어 후손 블록에도 동일한 규칙이 적용될 수 있습니다. 문단 시작을 강조하거나 가독성을 높이는 데 활용됩니다.
white-space .example5 { white-space: nowrap; }
줄바꿈 없이 이어지는 텍스트 예제
 공백과 줄바꿈 처리 방식을 지정합니다. nowrap은 줄바꿈을 막아 긴 텍스트가 한 줄로 표시됩니다. pre, pre-wrap 등 다른 값으로 입력 공백을 그대로 유지할 수도 있습니다.
text-transform .example6 { text-transform: uppercase; }
대문자로 변환된 텍스트
 텍스트를 대문자·소문자·첫 글자 대문자로 변환합니다. 상속되어 후손 텍스트에도 일괄 적용되며, 제목·버튼 등 강조에 자주 쓰입니다.
letter-spacing .example7 { letter-spacing: 4px; }
글자 간격 넓힘
 글자 사이 간격을 조절합니다. 상속되어 자식 텍스트에도 동일하게 적용됩니다. 디자인적으로 강조하거나 가독성을 조절할 때 사용합니다.
word-spacing .example8 { word-spacing: 20px; }
단어 간격 넓힘 예제
 단어 사이 간격을 조절합니다. 상속되어 자식 텍스트에도 동일하게 적용됩니다. 문단의 리듬을 바꾸거나 디자인적 효과를 줄 때 활용됩니다.
line-height .example9 { line-height: 2; }
줄 간격이 넓어진 텍스트
 줄 높이를 지정해 텍스트 블록 전체의 리듬을 만듭니다. 상속되어 후손 텍스트에도 적용되며, 가독성과 레이아웃 안정성에 큰 영향을 줍니다.
