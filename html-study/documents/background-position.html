CSS background-position 속성 이해하기

background-position은 요소의 배경 이미지가 어디에 위치할지를 결정하는 CSS 속성입니다.

값 설명 코드 예제 데모
left top 배경 이미지를 왼쪽 상단에 위치 
background-position: left top;
center 배경 이미지를 가운데에 위치 
background-position: center;
right bottom 배경 이미지를 오른쪽 하단에 위치 
background-position: right bottom;
30px 20px 배경 이미지를 x=30px, y=20px 위치에 배치 
background-position: 30px 20px;
80% 50% 배경 이미지를 요소 너비의 80%, 높이의 50% 위치에 배치 
background-position: 80% 50%;
