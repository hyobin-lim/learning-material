CSS Display: Block, Inline, Inline-block 정리

1. 개념

  • Block 요소: 한 줄 전체를 차지하며, 항상 새로운 줄에서 시작. div, p, h1~h6 등이 기본 block.
  • Inline 요소: 콘텐츠 크기만큼만 차지하며, 줄 안에서 다른 요소와 나란히 배치. span, a, strong 등이 기본 inline.
  • Inline-block 요소: 줄 안에서 나란히 배치되지만, block처럼 width, height, margin, padding을 자유롭게 적용 가능.

2. 특징 비교

구분 Block Inline Inline-block
줄 차지 한 줄 전체 차지 콘텐츠 크기만큼만 차지 콘텐츠 크기만큼 차지 (옆에 배치 가능)
줄바꿈 항상 줄바꿈 발생 줄바꿈 없음 줄바꿈 없음
width/height 적용 적용 가능 적용 불가 (콘텐츠 크기에 맞음) 적용 가능
margin/padding 상하좌우 모두 적용 좌우만 적용, 상하는 제한적 상하좌우 모두 적용
대표 태그 div, p, h1~h6 span, a, strong 기본은 없음, display: inline-block으로 지정

4. 핵심 요약

  • Block: 줄 전체 차지, 줄바꿈 발생, 크기 지정 가능
  • Inline: 콘텐츠 크기만큼, 줄 안에서 배치, 크기 지정 불가
  • Inline-block: 줄 안에서 배치 + 크기 지정 가능 → 두 속성의 장점 결합
