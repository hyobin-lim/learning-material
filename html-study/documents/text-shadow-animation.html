CSS text-shadow 애니메이션 효과

text-shadow는 @keyframes와 함께 사용하면 다양한 애니메이션 효과를 줄 수 있습니다.

효과 설명 CSS 코드 실제 표시
깜빡임(blink) 빨간 그림자가 깜빡이며 나타났다 사라지는 효과 @keyframes blink { ... }
animation: blink 1s infinite;		 Blink Text
네온사인(neon) 파란색과 노란색 네온사인처럼 색상이 번쩍이는 효과 @keyframes neon { ... }
animation: neon 2s infinite;		 Neon Text
흔들림(moveShadow) 그림자가 네 방향으로 움직이며 흔들리는 효과 @keyframes moveShadow { ... }
animation: moveShadow 3s infinite;		 Moving Shadow

정리: text-shadow는 단순한 그림자뿐 아니라 애니메이션과 결합해 깜빡임, 네온, 흔들림 같은 다양한 시각 효과를 만들 수 있습니다.
