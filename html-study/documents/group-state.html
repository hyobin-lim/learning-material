상태 제어 그룹

animation-fill-mode (hover 시 실행)

애니메이션이 시작 전·종료 후 어떤 스타일을 유지할지 지정합니다. 마우스를 올리면 실행되고, 종료 후 상태가 유지됩니다. 기본값은 none.

값 설명 예제 코드 데모
none 시작 전·종료 후 원래 상태 유지 
.demo.none { animation: moveBox 2s 1; animation-fill-mode: none; animation-play-state: paused; }
.demo.none:hover { animation-play-state: running; }
forwards 종료 후 마지막 상태 유지 
.demo.forwards { animation: moveBox 2s 1; animation-fill-mode: forwards; animation-play-state: paused; }
.demo.forwards:hover { animation-play-state: running; }
backwards 시작 전 첫 상태 적용 
.demo.backwards { animation: moveBox 2s 1; animation-fill-mode: backwards; animation-play-state: paused; }
.demo.backwards:hover { animation-play-state: running; }
both 시작 전 첫 상태 + 종료 후 마지막 상태 유지 
.demo.both { animation: moveBox 2s 1; animation-fill-mode: both; animation-play-state: paused; }
.demo.both:hover { animation-play-state: running; }

animation-play-state

애니메이션의 실행·일시정지를 제어합니다. 재생 중인 애니메이션에 적용됩니다.

값 설명 예제 코드 데모
running 애니메이션 실행 상태 
.box { animation: fadeBox 2s infinite; animation-play-state: running; }
paused 애니메이션 일시정지 상태.
hover를 뗐을 때 애니메이션이 일시정지 상태로 돌아감		 
.demo.paused { animation: fadeBox 2s infinite; animation-play-state: paused; }
hover로 실행 hover 시 실행 / 기본은 정지 
.box { animation: fadeBox 2s infinite; animation-play-state: paused; } 
.box:hover { animation-play-state: running; }

정리: 상태 제어 그룹은 애니메이션의 시작 전·종료 후 상태 유지와 실행·일시정지를 제어합니다.
