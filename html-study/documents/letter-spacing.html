CSS letter-spacing 속성 정리

letter-spacing은 텍스트의 글자 간격을 조절하는 속성입니다.

값 설명 예시 코드 실제 표시
normal 기본 글자 간격 letter-spacing: normal; Hello World
5px 글자 간격을 5px 넓힘 letter-spacing: 5px; Hello World
-2px 글자 간격을 좁힘 letter-spacing: -2px; Hello World
0.3em em 단위로 글자 간격 지정 letter-spacing: 0.3em; Hello World

정리: letter-spacing은 글자 간격을 넓히거나 좁혀서 가독성이나 디자인 효과를 줄 수 있으며, px, em, rem 단위를 활용해 유연하게 조절할 수 있습니다.


✨ 핵심 포인트

normal: 기본값

양수 값 → 글자 간격 넓어짐

음수 값 → 글자 간격 좁아짐

em, rem 단위 → 폰트 크기에 비례한 간격
