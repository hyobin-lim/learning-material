CSS 텍스트 꾸밈 속성 정리

텍스트를 꾸밀 때 자주 사용하는 속성들을 설명 → 예제 코드 → 데모로 정리했습니다.

텍스트 꾸밈 속성
속성 설명 예제 코드 데모
text-shadow 텍스트에 그림자를 추가합니다. (x, y, blur, color) text-shadow: 2px 2px 4px gray; 그림자 효과 텍스트
text-decoration 밑줄 표시 text-decoration: underline; 밑줄 텍스트
윗줄 표시 text-decoration: overline; 윗줄 텍스트
취소선 표시 text-decoration: line-through; 취소선 텍스트
꾸밈 제거 text-decoration: none; 꾸밈 없는 텍스트
text-transform 대문자로 변환 text-transform: uppercase; uppercase demo
소문자로 변환 text-transform: lowercase; LOWERCASE DEMO
첫 글자 대문자 text-transform: capitalize; capitalize demo text
letter-spacing 글자 간격을 조절합니다. letter-spacing: 4px; 간격 텍스트
word-spacing 단어 간격을 조절합니다. word-spacing: 12px; 단어 간격 데모 텍스트
