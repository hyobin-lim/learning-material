애니메이션 단축 속성 그룹

animation 단축 속성은 여러 애니메이션 관련 속성을 한 줄로 묶어 작성할 수 있습니다.
순서는 name: 키프레임 이름 → duration: 실행 시간 → timing-function: 속도 곡선 → delay: 지연 시간 → iteration-count: 반복 횟수 → direction: 방향 → fill-mode: 시작·종료 상태 유지 (+ 필요 시 play-state(실행/일시정지 상태)) 입니다.

예제

속성설명단축 코드데모
name 애니메이션 이름 
animation: moveBox 2s ease-in-out 0.5s 2 alternate forwards;
duration지속 시간 (2s)
timing-function속도 곡선 (ease-in-out)
delay시작 지연 (0.5s)
iteration-count반복 횟수 (2회)
direction재생 방향 (alternate)
fill-mode종료 후 상태 유지 (forwards)

정리: 단축 속성을 사용하면 여러 애니메이션 속성을 한 줄로 간결하게 작성할 수 있습니다. 다만 순서를 지켜야 하며, 일부 속성을 생략하면 기본값이 적용됩니다.
