CSS 상속 속성 - 실무 팁 (practical-tips)

속성 코드 예제 데모 설명
컨테이너 수준에서 설정 .container-demo {
  color: darkblue;
  font-family: "Courier New", monospace;
  line-height: 2;
  text-align: center;
}
이 컨테이너 안의 모든 텍스트는
동일한 스타일을 상속받습니다.
상단 래퍼에서 지정하면 하위 전체에 일관성 있게 퍼집니다.
블록 가운데 정렬 오해 방지 .block-demo { width: 200px; margin: 0 auto; }
이 블록은 가운데 정렬됩니다.
 블록 요소를 가로 가운데에 두려면 margin: 0 auto;를 사용해야 합니다. text-align: center는 인라인 콘텐츠에만 효과적입니다.
국소 재정의 .override-demo { color: darkred; font-style: italic; }
기본 텍스트 여기는 국소적으로 재정의된 텍스트
특정 섹션만 다른 타이포그래피/정렬이 필요하면 해당 섹션에만 속성을 재정의하세요. 필요 이상으로 하위에 강제하면 유지보수가 어려워집니다.
margin: auto .margin-auto { width: 200px; margin: 0 auto; }
가로 가운데 정렬된 블록
 블록 요소를 부모 안에서 가로 가운데 배치할 때 가장 많이 쓰이는 방법입니다.
padding .padding-demo { padding: 20px; }
내부 여백이 적용된 박스
 내용과 테두리 사이의 내부 여백을 지정합니다. 가독성과 디자인 균형을 맞출 때 중요합니다.
box-sizing .boxsizing-demo { box-sizing: border-box; }
패딩과 테두리를 포함한 크기 계산
 요소의 총 너비·높이에 패딩과 테두리를 포함시켜 레이아웃 계산을 단순화합니다. 실무에서 거의 필수적으로 사용됩니다.
overflow .overflow-demo { overflow: auto; }
긴 텍스트가 넘치면 스크롤바가 생깁니다. 긴 텍스트가 넘치면 스크롤바가 생깁니다.
 부모 컨테이너를 벗어나는 콘텐츠 처리 방식을 지정합니다. 스크롤 생성이나 잘림 여부를 제어합니다.
text-align .textalign-demo { text-align: center; }
가운데 정렬된 텍스트
 인라인 콘텐츠의 가로 정렬을 제어합니다. 버튼이나 제목 가운데 정렬에 자주 사용됩니다.
line-height .lineheight-demo { line-height: 2; }
줄 간격이 넓어진 텍스트
 줄 높이를 지정해 텍스트 블록의 가독성을 높입니다. 문단 전체의 리듬을 조절할 수 있습니다.
color .color-demo { color: darkblue; }
파란색 텍스트
 텍스트 색상을 지정합니다. 상속되어 자식 요소에도 동일하게 적용되어 문서 전체 톤을 맞출 수 있습니다.
font-family .fontfamily-demo { font-family: "Courier New", monospace; }
Courier New 글꼴 적용
 글꼴을 지정합니다. 상속되어 후손 텍스트에도 적용되며, 문서의 분위기와 가독성을 결정합니다.
