태그의 분류

태그가 웹페이지에서 차지하는 영역에 따라서 두 가지로 나눌 수 있다.

특징 블록레벨 (Block Level) 인라인레벨 (Inline Level)
시작 시점 항상 새로운 줄에서 시작 새로운 줄에서 시작하지 않고, 줄 안에서 이어짐
차지하는 공간 웹페이지의 화면 전체 너비를 차지 필요한 콘텐츠의 너비만큼만 차지
사용 시점
  • 문서의 구조를 나눌 때, 큰 레이아웃이나 구획을 정의할 때
  • 제목, 문단, 목록, 표 등 독립적인 콘텐츠 단위를 표현할 때
  • 텍스트 일부를 꾸미거나 강조할 때
  • 문장 중간에 링크, 이미지, 입력 필드 등을 삽입할 때
상속 여부 다른 블록 요소 (중첩 가능)와 인라인 요소를 포함할 수 있음 블록 요소 안에서 사용되며 다른 인라인 요소와 나란히 배치 가능
줄 바꿈 태그 앞뒤로 자동 줄바꿈 발생 줄바꿈이 없음 (한 줄에 여러 개 배치 가능)
너비/높이 설정 width, height 속성 적용 가능 width, height 속성 적용 불가 (대신 display:inline-block 사용)
위아래 여백 margin-top, margin-bottom 적용 가능 margin-top, margin-bottom 적용 불가
예시 태그
  • <div>, <p>, <h1>~<h6>, <section>, <article>
  • <ul>, <ol>, <li>, <table>, <form>
  • <span>, <strong>, <em>, <a>
  • <img>, <br>, <input>, <label>

특별한 태그 설명

<div> 태그 (블록 레벨)

  • 의미: Division의 약자로 '구획'을 의미함
  • 용도: 웹 페이지를 여러 섹션으로 나누기 위한 컨테이너
  • 특징: 특정한 의미가 없고, 단순히 그룹화할 때 사용
  • CSS 적용: 폭, 높이, 배경색, 패딩, 마진 등 모두 적용 가능

<span> 태그 (인라인 레벨)

  • 의미: 텍스트의 일부를 묶어서 스타일링할 때 사용
  • 용도: 특정 단어나 구절에 색상, 폰트 변화 등을 주기 위해 사용
  • 특징: 특정한 의미가 없고, 단순히 인라인 스타일링 목적
  • CSS 적용: 폭, 높이는 적용 불가, 색상, 글씨 크기 등만 적용 가능
