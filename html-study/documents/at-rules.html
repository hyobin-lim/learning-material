CSS @ 규칙 (At-rules) 정리

CSS에서 @는 특별한 지시문을 나타내며, 스타일 시트의 동작을 제어하거나 조건부 규칙을 정의할 때 사용됩니다.

@는 링크를 연결하는 것이므로 링크 주소를 실행시켜 제대로 연결이 되어 있는지 오류가 나지 않았는지 확인하는 습관이 필요합니다.

@ 규칙 설명 예시 코드
@import 다른 CSS 파일을 불러올 때 사용. 반드시 스타일 규칙보다 위에 있어야 함.
문서에 css 문서를 삽입하는 역할을 합니다.		 @import url("style.css");
@font-face 사용자 정의 글꼴을 등록합니다.
웹폰트를 정의하여 문서에서 사용할 수 있게 함.
파일과 css문서를 연결시켜주는 역할을 합니다.		 @font-face {
  font-family: "MyFont";
  src: url("MyFont.woff2") format("woff2");
}
@media 조건부 스타일 적용. 반응형 디자인에 사용
미디어 쿼리: 화면 크기나 장치 조건에 따라 다른 스타일을 적용합니다.		 @media (max-width: 600px) {
  p { font-size: 14px; color: red; }
}
@keyframes CSS 애니메이션의 단계별 동작을 정의합니다. animation 속성과 함께 사용. @keyframes blink {
  0% { opacity: 1; }
  50% { opacity: 0; }
  100% { opacity: 1; }
}
@supports 브라우저가 특정 CSS 기능을 지원하는지 검사합니다. @supports (display: grid) { div { display: grid; } }
@page 인쇄 시 페이지 레이아웃을 정의합니다. @page { margin: 2cm; }

정리: @import는 외부 CSS 불러오기, @font-face는 폰트 정의, @media는 조건부 스타일, @keyframes는 애니메이션을 정의하는 데 사용됩니다.
