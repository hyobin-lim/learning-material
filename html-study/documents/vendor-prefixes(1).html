자주 쓰이는 CSS 벤더 프리픽스

최신 브라우저에서는 대부분 표준 속성으로 동작하지만, 일부 속성은 여전히 벤더 프리픽스가 필요합니다.

-webkit-line-clamp

설명코드 예제데모
텍스트를 지정한 줄 수까지만 표시하고 나머지는 잘라냄 
.line-clamp {
  display: -webkit-box;
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;
  overflow: hidden;
}
이 텍스트는 여러 줄로 작성되어 있습니다.
하지만 -webkit-line-clamp 속성을 사용하면
최대 3줄까지만 표시되고
그 이후 내용은 잘려서 보이지 않습니다.
추가 텍스트가 있어도 표시되지 않습니다.

-webkit-background-clip: text

설명코드 예제데모
텍스트 모양에 맞게 배경을 잘라내어 표시 
.clip-text {
  background: linear-gradient(45deg, #ff7f50, #ff1493);
  -webkit-background-clip: text;
  color: transparent;
}
Gradient Text

Tip: -webkit- 접두사는 크롬, 사파리, iOS 브라우저 등 WebKit 기반 브라우저에서 주로 필요합니다.
