CSS text-shadow 속성 정리

text-shadow는 텍스트에 그림자를 주어 입체감이나 강조 효과를 줄 수 있습니다.

text-shadow: offset-x offset-y blur-radius color;

offset-x: 그림자의 가로 위치 (양수 → 오른쪽, 음수 → 왼쪽)

offset-y: 그림자의 세로 위치 (양수 → 아래, 음수 → 위)

blur-radius: 그림자의 흐림 정도 (선택 사항)

color: 그림자의 색상

예시 설명 CSS 코드 실제 표시
기본 그림자 오른쪽 아래로 흐린 회색 그림자 text-shadow: 2px 2px 2px gray; Text Shadow
반대 방향 그림자 왼쪽 위로 붉은 그림자 text-shadow: -2px -2px 4px red; Text Shadow
투명 그림자 반투명 검정 그림자 text-shadow: 4px 4px 6px rgba(0,0,0,0.5); Text Shadow
광채 효과 파란색 빛나는 효과 text-shadow: 0 0 10px blue; Text Shadow
다중 그림자 검정과 주황 그림자를 동시에 적용 text-shadow: 2px 2px 2px black, -2px -2px 2px orange; Text Shadow

정리: text-shadow는 위치, 흐림 정도, 색상을 조합해 다양한 그림자 효과를 줄 수 있으며, 여러 개의 그림자를 동시에 적용할 수도 있습니다.
