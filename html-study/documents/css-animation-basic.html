CSS 애니메이션 기초와 전체 개요

애니메이션이란?

CSS 애니메이션은 요소의 스타일을 일정 시간 동안 변화시켜 동적인 효과를 줄 수 있는 기능입니다.
@keyframes 규칙으로 스타일 변화 과정을 정의하고, 요소에 animation 속성을 적용해 시간 흐름에 따라 변화를 재생합니다.

전체 설명

애니메이션 속성은 개별 속성과 단축 속성으로 구성됩니다. 개별 속성은 각 개념을 독립적으로 제어하고, 단축 속성은 여러 값을 한 줄로 묶어 선언합니다.
실무에서는 가독성과 유지보수를 위해 상황에 따라 두 방식을 혼용합니다.

핵심 개념

  • @keyframes: 애니메이션의 단계별 스타일 변화를 정의
  • animation 속성들: 이름, 시간, 속도 곡선, 반복 횟수, 방향, 상태 유지 등을 세밀하게 제어
  • 단축 속성: 여러 값을 한 줄로 묶어 선언 가능

학습 순서

아래 순서대로 학습하면 CSS 애니메이션 속성을 체계적으로 이해할 수 있습니다:

개별 속성

각각 독립적으로 애니메이션의 특정 동작을 제어

  1. 기본 제어 그룹 — 애니메이션의 이름, 실행 시간, 속도 곡선, 지연 시간을 제어합니다.
  2. 반복·방향 그룹 — 애니메이션의 반복 횟수와 재생 방향을 설정합니다.
  3. 상태 제어 그룹 — 애니메이션 시작 전·종료 후 상태 유지와 실행/일시정지를 제어합니다.
  4. 실험적 속성 그룹 — 최신 브라우저에서만 지원되는 실험적 속성들을 다룹니다.

단축 속성

animation 하나로 여러 개별 속성을 한 줄에 묶어 선언

  1. 단축 속성 그룹 — 여러 속성을 한 줄로 묶어 선언하는 단축 속성을 설명합니다.
