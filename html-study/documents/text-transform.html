CSS text-transform 속성 정리

text-transform은 텍스트의 대소문자 변환을 제어합니다.

값 설명 예시 코드 실제 표시
none 변환 없음 (원래 입력된 그대로 표시) text-transform: none; Hello World
uppercase 모든 글자를 대문자로 변환 text-transform: uppercase; Hello World
lowercase 모든 글자를 소문자로 변환 text-transform: lowercase; Hello World
capitalize 각 단어의 첫 글자를 대문자로 변환 text-transform: capitalize; hello world

정리: text-transform은 텍스트의 대소문자 스타일을 손쉽게 바꿀 수 있으며, 제목이나 버튼 텍스트를 꾸밀 때 자주 사용됩니다.
