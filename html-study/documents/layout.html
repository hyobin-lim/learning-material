레이아웃(Layout) 완전 정리

1. 레이아웃이란?

레이아웃(Layout)은 웹 페이지나 디자인에서 콘텐츠(텍스트, 이미지, 버튼 등)를 화면 안에 어떻게 배치할지에 대한 구조와 구성 방식을 의미합니다.
즉, 사용자가 정보를 쉽게 이해하고 탐색할 수 있도록 화면을 영역별로 나누고 질서를 만드는 것입니다.

2. 웹 디자인 관점에서의 레이아웃

  • 헤더(Header): 상단 영역, 로고·메뉴·검색창 등이 위치
  • 내비게이션(Navigation): 사이트 이동을 위한 메뉴
  • 메인(Main Content): 핵심 콘텐츠가 들어가는 중심 영역
  • 사이드바(Sidebar): 보조 정보, 광고, 링크 등이 들어가는 옆 영역
  • 푸터(Footer): 하단 영역, 저작권·연락처·추가 링크 등이 위치

3. CSS 코드 관점에서의 레이아웃

CSS 코드 관점에서의 레이아웃(Layout)은 웹 페이지의 요소들을 화면에 어떻게 배치하고, 크기를 계산하며, 서로의 관계를 정의하는지를 결정하는 규칙과 속성들의 집합입니다.
즉, HTML로 구조를 만들고 CSS로 그 구조를 어떻게 보이게 할지를 설계하는 것입니다.

6. 핵심 요약

  • 레이아웃 = 콘텐츠를 화면에 배치하는 구조와 방식
  • 웹 디자인 관점에서는 헤더·내비게이션·메인·사이드바·푸터로 구성
  • CSS 코드 관점에서는 float, position, flexbox, grid, media query 등을 활용
  • Flexbox는 1차원, Grid는 2차원 레이아웃에 적합
  • 반응형 웹에서는 Media Query와 Grid/Flexbox를 함께 사용
