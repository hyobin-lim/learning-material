섹션3 - HTML 폰트 연결

HTML과 CSS는 다양한 방식으로 폰트를 연결할 수 있습니다.

폰트 연결 방법
방식 설명 예제 코드 표시 모습
@font-face 직접 서버에 폰트 파일을 두고 불러오기 
@font-face {
  font-family: 'MyFont';
  src: url('myfont.woff2') format('woff2'),
       url('myfont.woff') format('woff');
  font-display: swap;
}
 이 문장은 커스텀 폰트로 보입니다.
Google Fonts 외부 구글 폰트 서비스에서 불러오기 
<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
 이 문장은 Roboto 폰트로 보입니다.
@import CSS에서 @import로 외부 폰트 불러오기 
@import url('https://fonts.googleapis.com/css2?family=Indie+Flower&display=swap');
 이 문장은 손글씨 느낌의 Indie Flower 폰트로 보입니다.
시스템 기본 폰트 운영체제 기본 폰트 사용 
body {
  font-family: system-ui, sans-serif;
}
 이 문장은 운영체제 기본 폰트로 보입니다.
변수 폰트 가변 폰트의 축을 조절해 다양한 스타일 제공 
body {
  font-family: 'Roboto Flex', sans-serif;
  font-variation-settings: 'wght' 700;
}
 이 문장은 Roboto Flex 폰트로 굵게 보입니다.
font-display 옵션 폰트 로딩 시 표시 방식 제어 
@font-face {
  font-family: 'MyFont';
  src: url('myfont.woff2') format('woff2');
  font-display: fallback;
}
 이 문장은 font-display 옵션 적용 폰트로 보입니다.
한글 웹폰트 한국어 환경에서 자주 쓰이는 웹폰트 
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+KR&display=swap" rel="stylesheet">
 이 문장은 Noto Sans KR 폰트로 보입니다.
성능 최적화 preconnect로 폰트 서버 연결 최적화 
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
 이 문장은 preconnect로 빠르게 로드된 Roboto 폰트로 보입니다.
변수 폰트 확장 굵기(wght), 너비(wdth) 등 여러 축 조절 
body {
  font-family: 'Roboto Flex', sans-serif;
  font-variation-settings: 'wght' 700, 'wdth' 120;
}
 이 문장은 Roboto Flex 폰트로 굵고 넓게 보입니다.
시스템 폰트 스택 확장 다양한 OS 환경에서 기본 폰트 지정 
body {
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI',
               Roboto, 'Helvetica Neue', Arial, sans-serif;
}
 이 문장은 각 운영체제 기본 폰트로 보입니다.
