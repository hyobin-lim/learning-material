CSS 상태 선택자 (Pseudo-class) 정리

상태 선택자는 요소가 특정 상태일 때 스타일을 적용할 수 있게 해주는 CSS 선택자입니다.

선택자 설명 예시
:link 아직 방문하지 않은 링크 a:link { color: blue; }
:visited 방문한 링크 a:visited { color: purple; }
:hover 마우스를 올렸을 때 a:hover { text-decoration: underline; }
:active 클릭 순간 a:active { color: red; }
:focus 포커스를 가진 입력 요소 input:focus { border: 2px solid blue; }
:checked 체크된 체크박스/라디오 버튼 input:checked { background: yellow; }
:disabled 비활성화된 입력 요소 input:disabled { color: gray; }
:first-child 부모의 첫 번째 자식 li:first-child { font-weight: bold; }
:last-child 부모의 마지막 자식 li:last-child { color: green; }
:nth-child(n) 부모의 n번째 자식 li:nth-child(2) { color: red; }

정리: 상태 선택자는 링크, 폼 요소, 구조적 위치 등 다양한 상황에서 스타일을 지정할 수 있습니다.
