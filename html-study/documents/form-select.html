Form 섹션 3 - 선택 요소

전체 설명

선택 요소(Selection Elements)는 사용자가 **미리 준비된 여러 옵션 중에서 하나 또는 여러 개를 고를 수 있도록 제공하는 입력 방식**입니다.
직접 값을 입력하는 것이 아니라, **라디오 버튼, 체크박스, 드롭다운 목록** 같은 UI를 통해 선택합니다.
따라서 선택 요소는 사용자가 빠르고 정확하게 데이터를 고를 수 있도록 도와줍니다.

상세 설명

<input type="radio"> (라디오 버튼)은 같은 name (이름)으로 묶으면 그룹 내에서 하나만 선택됩니다.
<input type="checkbox"> (체크박스)는 여러 개를 동시에 선택할 수 있습니다.
<select> (드롭다운 목록)은 옵션을 펼쳐서 선택할 수 있으며, multiple (다중 선택)과 size (보이는 항목 수) 속성으로 확장된 UI를 제공합니다.
<optgroup> (옵션 그룹)은 옵션들을 카테고리별로 묶어줍니다.

자주 쓰는 속성들:
• checked (체크됨): 라디오/체크박스 기본 선택 상태
• selected (선택됨): select 옵션 기본 선택 상태
• disabled (비활성화): 선택 불가 상태
• value (값): 서버로 전송되는 실제 데이터 값
항목설명예제 코드데모
radio (라디오 버튼) 단일 선택, 같은 name 그룹 내에서 하나만 선택 가능 
<input type="radio" name="color" value="red" checked> 빨강
checkbox (체크박스) 다중 선택 가능, 여러 값이 배열처럼 전송됨 
<input type="checkbox" name="hobby" value="music" checked> 음악
select (드롭다운) 옵션 중 하나 선택, selected로 기본값 지정 가능 
<select name="city">
  <option value="">선택</option>
  <option value="seoul" selected>서울</option>
  <option value="busan">부산</option>
</select>
select multiple (다중 선택) 여러 항목을 동시에 선택, size로 보이는 항목 수 지정 
<select name="skills" multiple size="4">
  <option>HTML</option>
  <option>CSS</option>
  <option>JavaScript</option>
  <option>Python</option>
</select>
optgroup (옵션 그룹) 옵션들을 그룹화하여 카테고리별로 구분 
<select>
  <optgroup label="과일">
    <option>사과</option>
    <option>바나나</option>
  </optgroup>
  <optgroup label="야채">
    <option>당근</option>
    <option>양상추</option>
  </optgroup>
</select>
