css float 속성 정리 css float property guide

예제 코드

.box.left { float: left; }
.box.right { float: right; }
.box.none { float: none; }
.box.inline-start { float: inline-start; }
.box.inline-end { float: inline-end; }
.clearfix::after { content:""; display:block; clear:both; }

none은 띄우지 않고 기본 흐름에 따라 아래쪽으로 배치됩니다.

쉬운 설명

float은 상자를 물 위에 띄워서 왼쪽이나 오른쪽으로 밀어놓는 것과 같아요.
책상 위에 종이를 왼쪽으로 밀어놓으면 다른 종이가 오른쪽 공간에 놓이는 것과 비슷합니다.
float:none을 쓰면 종이를 그냥 책상 위에 두는 것처럼, 기본 흐름(위→아래)에 따라 배치돼요.
inline-start와 inline-end는 글쓰기 방향에 따라 자동으로 왼쪽/오른쪽을 결정해줘서
아랍어처럼 오른쪽에서 왼쪽으로 쓰는 언어에서도 올바르게 배치할 수 있습니다.
마지막으로 clear 속성은 "다음 줄은 float된 요소 옆에 붙지 말고, 그 아래로 내려가라"는 의미예요.
