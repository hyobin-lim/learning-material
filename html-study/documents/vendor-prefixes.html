CSS 벤더 프리픽스 정리

벤더 프리픽스(Vendor Prefix)는 브라우저별로 CSS 최신 기능을 실험적으로 지원할 때 붙이는 접두사입니다.

접두사 대상 브라우저 예제 코드 데모
-webkit- Chrome, Safari, iOS 브라우저 
-webkit-border-radius: 20px;
WebKit
-moz- Firefox 
-moz-border-radius: 20px;
Mozilla
-ms- Internet Explorer 
-ms-transform: rotate(10deg);
MS IE
-o- 옛 Opera (Presto 엔진) 
-o-border-radius: 20px;
Opera

Tip: 최신 브라우저에서는 대부분 표준 속성으로 동작하기 때문에 벤더 프리픽스가 필요 없는 경우가 많습니다. 하지만 일부 속성(-webkit-line-clamp, -webkit-background-clip: text 등)은 여전히 접두사가 있어야 동작합니다.
