CSS text-decoration 속성 정리

text-decoration 속성은 텍스트에 밑줄, 윗줄, 취소선 등 다양한 꾸밈을 적용할 수 있습니다.
또한 선의 스타일(solid, dotted, dashed, double, wavy)과 색상도 지정할 수 있습니다.

text-decoration 속성 예제
속성 값 설명 예제 코드 데모
underline 밑줄 표시 text-decoration: underline; 밑줄 텍스트
overline 윗줄 표시 text-decoration: overline; 윗줄 텍스트
line-through 취소선 표시 text-decoration: line-through; 취소선 텍스트
none 꾸밈 제거 text-decoration: none; 꾸밈 없는 텍스트
underline wavy red 물결 모양 밑줄 + 색상 text-decoration: underline wavy red; 물결 밑줄 텍스트
underline double blue 이중 밑줄 + 색상 text-decoration: underline double blue; 이중 밑줄 텍스트
underline dotted green 점선 밑줄 + 색상 text-decoration: underline dotted green; 점선 밑줄 텍스트
