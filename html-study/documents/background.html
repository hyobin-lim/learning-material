CSS background-* 속성 종합 정리

아래 표는 background 관련 모든 주요 속성을 설명과 코드, 데모로 정리한 것입니다.

background-color

설명 코드 데모
배경 색상 지정 
background-color: lightgreen;

background-image

설명 코드 데모
배경 이미지 지정 
background-image: url("../images/book-icon.png");

background-repeat

값 설명 코드 데모
repeat 가로·세로 반복 
background-repeat: repeat;
no-repeat 반복하지 않음 
background-repeat: no-repeat;
repeat-x 가로 반복 
background-repeat: repeat-x;
repeat-y 세로 반복 
background-repeat: repeat-y;

background-position

값 설명 코드 데모
left top 왼쪽 상단 
background-position: left top;
center 가운데 
background-position: center;
right bottom 오른쪽 하단 
background-position: right bottom;

background-attachment

값 설명 코드 데모
scroll 기본값. 페이지 스크롤 시 배경도 함께 이동 
background-attachment: scroll;
스크롤을 내려 보세요. (scroll)
배경은 요소 내부 스크롤과 함께 움직입니다.

background-size

값 설명 코드 데모
auto 원래 크기 유지 
background-size: auto;
cover 요소 전체 덮기 
background-size: cover;
contain 요소 안에 이미지 포함 
background-size: contain;

background-origin

값 설명 코드 데모
border-box 테두리 기준으로 테두리 왼쪽 상단에서 시작 
background-origin: border-box;
padding-box 패딩 기준으로 패딩 왼쪽 상단에서 시작 
background-origin: padding-box;
content-box 콘텐츠 기준으로 콘텐츠 중앙에서 시작 
background-origin: content-box;

background-clip

값 설명 코드 예제 데모
border-box 배경이 테두리까지 채워짐 
background-clip: border-box;
내용
padding-box 배경이 패딩 영역까지만 채워짐 
background-clip: padding-box;
내용
content-box 배경이 콘텐츠 영역까지만 표시됨 
background-clip: content-box;
내용
text 배경이 텍스트 모양에 맞게 잘려 표시됨 
background-clip: text;
         color: transparent;
