CSS 표 스타일 정리

표는 border, padding, background, text-align 등을 활용해 꾸밀 수 있습니다.

스타일 속성 설명 예시 코드
border 셀 테두리 지정 border: 1px solid #ccc;
border-collapse 테두리 겹침 여부 border-collapse: collapse;
padding 셀 안쪽 여백 padding: 12px;
text-align 텍스트 정렬 (left, center, right) text-align: center;
background 셀 배경색 지정 background: #f2f2f2;
nth-child 짝수/홀수 행 스타일 지정 tr:nth-child(even) { background: #f2f2f2; }
hover 마우스 오버 시 강조 tr:hover { background: #dfe6e9; }
