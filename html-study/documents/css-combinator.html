CSS 결합자(Combinator) 정리

CSS에서 결합자는 요소 간의 관계를 정의하여 스타일을 적용할 대상을 더 구체적으로 지정합니다. 결합자 자체는 특이성 점수(Specificity)를 높이지 않지만, 선택자를 더 구체적으로 만들어 캐스캐이딩에서 우선 적용될 수 있습니다.

결합자 종류와 예시

결합자 이름 설명 예시 적용 대상
공백 ( ) 자손 선택자 앞 요소의 모든 하위(자손) 요소 선택 div p { color: blue; } <div> 안의 모든 <p>
> 자식 선택자 앞 요소의 직계 자식만 선택 ul > li { color: red; } <ul> 바로 아래의 <li>
+ 인접 형제 선택자 앞 요소 바로 뒤에 오는 형제 요소 선택 h1 + p { color: green; } <h1> 바로 뒤의 <p>
~ 일반 형제 선택자 앞 요소 뒤에 오는 모든 형제 요소 선택 h1 ~ p { color: orange; } <h1> 뒤에 오는 모든 <p>

우선순위 설명

  • 결합자 자체는 특이성 점수를 올리지 않습니다.
  • 하지만 결합자를 사용하면 선택자가 더 구체적이 되어, 단순한 태그 선택자보다 우선 적용됩니다.
  • 예: div p (자손 선택자) vs p (태그 선택자) → 둘 다 특이성 점수는 같지만, div p가 더 구체적이므로 캐스캐이딩에서 우선 적용됩니다.

정리

  • 공백: 모든 하위 요소
  • >: 직계 자식만
  • +: 바로 뒤의 형제 하나
  • ~: 뒤에 오는 모든 형제
  • → 결합자는 선택 범위를 좁혀주며, 캐스캐이딩에서 더 구체적인 규칙으로 인정됩니다.
