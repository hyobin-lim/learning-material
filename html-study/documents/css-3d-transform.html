3D Transform 함수

CSS 3D Transform은 요소를 평면(2D)에서 벗어나 입체 공간으로 이동, 회전, 확대/축소할 수 있게 해줍니다.
특징: X·Y·Z 세 축을 모두 다룰 수 있으며, 원근(perspective)을 적용해 깊이감을 표현할 수 있습니다.
필수 조건: 부모 컨테이너에 perspective를 지정하고, 자식 요소에 transform-style: preserve-3d를 적용해야 합니다.
활용: 카드 뒤집기, 갤러리, 3D 애니메이션, 입체 UI 등 다양한 연출에 사용됩니다.

추가/확장 3D transform 함수

기본 3D 함수 외에 X·Y·Z를 동시에 다루거나 원근감을 직접 지정할 수 있는 확장형 함수들입니다.

함수 설명 예제 데모
translate3d() 지원 단위: px, %, em
포인트: X, Y, Z를 동시에 이동
원리: 3차원 좌표 이동
왜: 한 번에 3축 이동 가능
주의: perspective 컨테이너 필요
응용: 복잡한 3D 애니메이션 시작 위치 지정 
transform: translate3d(150px, 80px, 200px);

perspective() 지원 단위: px
포인트: 부모 컨테이너에 적용해 원근감을 지정

왜: 값이 작을수록 왜곡이 강하고, 값이 클수록 평면에 가까워짐.
주의: 부모 컨테이너에 적용해야 효과가 확실
응용: 3D 카드/갤러리 원근감 표현 
.scene-persp-near { perspective: 100px; }
.scene-persp-far  { perspective: 800px; }
