가상 선택자(Pseudo-selector)란?

가상 선택자는 문서의 실제 DOM 요소를 추가로 만들지 않고, **특정 상태**나 **요소의 일부**를 선택해 스타일을 적용하는 CSS 기능입니다. 크게 두 가지로 나뉩니다.

종류 설명 대표 예시
가상 클래스 (Pseudo-class) 요소가 특정 상태일 때 선택합니다. :hover, :active, :focus, :visited, :first-child, :nth-child(), :required, :valid, :invalid
가상 요소 (Pseudo-element) 요소의 특정 부분을 선택합니다. ::before, ::after, ::first-line, ::first-letter, ::selection

코드 예시

/* 가상 클래스 */
.button:hover { background: #cbd5e1; }
.list li:nth-child(2n) { background: #f8fafc; }
input:required { border-color: #ef4444; }

/* 가상 요소 */
.quote::before { content: "“"; }
.quote::after { content: "”"; }
.required-label::after { content: " *"; color: #ef4444; }

인용문 가상 요소

텍스트 부분 스타일

아래 카드의 첫 줄과 첫 글자는 다르게 보입니다.

first-line과 first-letter를 이용해 텍스트 일부만 스타일링합니다.

언제 사용하나요?

  • 상태 기반 스타일: 버튼의 hover/active/focus 상태, 입력의 valid/invalid 상태 표시
  • 장식/보강: ::before/::after로 아이콘, 구분점, 보조 텍스트 추가
  • 부분 선택: ::first-line, ::first-letter로 텍스트의 특정 부분만 강조

기억 포인트

  • 콜론 개수: 가상 클래스는 한 콜론(:), 가상 요소는 두 콜론(::)이 권장 표기
  • DOM 추가 없음: 가상 요소는 실제 노드가 생기는 게 아니라 렌더링 단계에서만 표현
  • 접근성: :focus 스타일을 제공해 키보드 사용자도 편하게
