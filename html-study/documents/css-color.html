CSS color 속성 정리 (OKLab 포함)

CSS의 color 속성은 텍스트 색상을 지정합니다. 다양한 표기법을 사용할 수 있으며, 최신 사양에서는 OKLab/OKLCH도 지원합니다.

표현 방식 설명 예시 코드 실제 표시
색상 이름 미리 정의된 색상 이름 사용 color: red; Red Text
HEX 코드 16진수로 색상 지정 (#RRGGBB) color: #00ff00; Green Text
RGB 빨강, 초록, 파랑 값(0~255) color: rgb(0,0,255); Blue Text
RGBA RGB + 투명도(Alpha, 0~1) color: rgba(255,0,0,0.5); Semi Red
HSL 색상(Hue), 채도(Saturation), 명도(Lightness) color: hsl(200,100%,40%); HSL Text
HSLA HSL + 투명도(Alpha) color: hsla(120,100%,25%,0.7); HSLA Text
OKLab 사람의 시각에 맞춘 균일 색 공간 (Lightness, a, b) color: oklab(60% 0.1 0.1); OKLab Text
OKLCH OKLab 기반의 색상 표현 (Lightness, Chroma, Hue) color: oklch(70% 0.2 40); OKLCH Text

정리: color 속성은 텍스트 색상을 지정하며, 다양한 표기법을 지원합니다. 상황에 따라 HEX, RGB, HSL 등을 선택해 사용하면 됩니다.
