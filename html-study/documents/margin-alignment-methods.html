margin을 활용한 정렬 방법

쉬운 설명

상자를 화면 정 가운데 놓고 싶을 때는 상자의 위치를 top:50%; left:50%으로 잡아요.
그러면 상자의 왼쪽 위 모서리가 화면 중앙에 오게 돼요.
하지만 우리가 원하는 건 상자의 중심이 화면 중앙에 오는 것이죠.
그래서 상자의 절반 크기만큼 위와 왼쪽으로 다시 빼줘야 합니다. 따라서 음수 마진을 주어 모서리 기준점을 상자의 중심 기준점으로 바꿔줍니다.
이렇게 하면 상자의 중심이 정확히 화면 중앙에 위치하게 돼요.
마치 책상 가운데에 블록을 놓고, 블록 크기만큼 살짝 밀어 맞추는 것과 같아요.

가로 중앙은 margin: 0 auto;를 쓰면 되고, 세로 중앙은 flex 컨테이너와 margin을 함께 쓰면 돼요.

예제 코드 (중앙 정렬)

.center-box {
  position: absolute;
  top: 50%;
  left: 50%;
  margin-top: -50px;  /* 요소 높이의 절반 */
  margin-left: -50px; /* 요소 너비의 절반 */
}
