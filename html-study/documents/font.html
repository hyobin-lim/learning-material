폰트 속성 정리

속성 설명 예제 코드 데모
font-size 텍스트 크기를 조절합니다. 
p { font-size: 24px; }

큰 글씨 크기
font-family 글꼴을 지정합니다. 
p { font-family: "Courier New", monospace; }

Courier New 글꼴
font-weight 글자의 굵기를 지정합니다. 
p { font-weight: bold; }

굵은 글씨
font-style 글자의 스타일을 지정합니다 (기울임 등). 
p { font-style: italic; }

기울임 글씨
text-transform: uppercase 텍스트를 모두 대문자로 변환합니다. 
p { text-transform: uppercase; }

uppercase example
text-transform: lowercase 텍스트를 모두 소문자로 변환합니다. 
p { text-transform: lowercase; }

LOWERCASE EXAMPLE
text-transform: capitalize 각 단어의 첫 글자를 대문자로 변환합니다. 
p { text-transform: capitalize; }

capitalize example text
color 텍스트 색상을 지정합니다. 
p { color: crimson; }
