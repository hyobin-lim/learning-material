Display: Flex 속성 정리

display:flex는 부모 요소를 플렉스 컨테이너로 만들어 자식들을 플렉스 아이템으로 배치하는 속성입니다.
이때 컨테이너와 아이템에 적용할 수 있는 다양한 속성이 함께 동작합니다.
핵심은 컨테이너 속성(정렬, 방향, 줄바꿈)과 아이템 속성(크기, 비율, 순서)으로 나뉘며, 각각 레이아웃과 크기 비율을 제어합니다.

flex-direction, flex-wrap, justify-content, align-items, align-content의 모든 값과 차이를 설명하고, 값별 데모를 제공합니다.

축 개념 정리: 주축(main axis)은 아이템이 나열되는 방향이고, 교차축(cross axis)은 주축에 수직인 방향입니다. flex-direction이 축을 결정합니다.
- row/row-reverse: 주축=가로, 교차축=세로
- column/column-reverse: 주축=세로, 교차축=가로

flex-direction

플렉스 컨테이너의 주축 방향을 설정합니다. 이 값에 따라 justify-content가 정렬하는 축과 align-items/align-content가 정렬하는 축이 달라집니다.

값 설명 예제 코드
row 기본값. 주축인 가로(왼쪽→오른쪽)로 아이템 배치. 교차축은 세로(위→아래). 
.container { display: flex; flex-direction: row; }
row-reverse 주축인 가로(오른쪽→왼쪽)로 아이템 배치. 교차축은 세로(위→아래). 
.container { display: flex; flex-direction: row-reverse; }
column 주축인 세로(위→아래)로 아이템 배치. 교차축은 가로(왼쪽→오른쪽). 
.container { display: flex; flex-direction: column; }
column-reverse 주축인 세로(아래→위)로 아이템 배치. 교차축은 가로(왼쪽→오른쪽). 
.container { display: flex; flex-direction: column-reverse; }

flex-wrap

아이템이 한 줄에 모두 배치될지, 공간이 부족하면 다음 줄로 감쌀지를 결정합니다.

값 설명 예제 코드
nowrap 기본값. 줄바꿈 없이 한 줄에 배치. 
.container { display: flex; flex-wrap: nowrap; }
wrap 공간이 부족하면 다음 줄로 감싸기. 
.container { display: flex; flex-wrap: wrap; }
wrap-reverse 줄 감싸기 방향을 반대로(줄 스택의 시작/끝이 뒤집힘). 
.container { display: flex; flex-wrap: wrap-reverse; }

justify-content

아이템을 주축 방향으로 어떻게 정렬할지 설정합니다. flex-direction에 따라 주축이 달라집니다.

값 설명 예제 코드
flex-start 주축 시작점에 모음. 
.container { justify-content: flex-start; }
flex-end 주축 끝점에 모음. 
.container { justify-content: flex-end; }
center 주축 중앙에 모음. 
.container { justify-content: center; }
space-between 양 끝에 붙이고 사이 간격을 동일하게. 
.container { justify-content: space-between; }
space-around 각 아이템 양쪽 여백을 동일하게. 
.container { justify-content: space-around; }
space-evenly 아이템 사이와 양 끝 간격을 모두 동일하게. 
.container { justify-content: space-evenly; }

align-items

아이템을 교차축 방향으로 어떻게 정렬할지 설정합니다. 각 아이템의 높이나 텍스트 기준선에 따라 차이가 납니다.

값 설명 예제 코드
stretch 기본값. 아이템을 교차축으로 가능한 범위까지 늘립니다(높이 자동일 때). 
.container { align-items: stretch; }
flex-start 교차축 시작점에 정렬. 
.container { align-items: flex-start; }
flex-end 교차축 끝점에 정렬. 
.container { align-items: flex-end; }
center 교차축 중앙에 정렬. 
.container { align-items: center; }
baseline 아이템들의 텍스트 기준선을 맞춤. 
.container { align-items: baseline; }

align-content

여러 줄로 감싼(=wrap된) 줄들 전체를 교차축 방향으로 어떻게 배치할지 설정합니다. 한 줄 레이아웃에서는 효과가 없습니다. flex-wrap이 wrap 또는 wrap-reverse일 때 의미가 있습니다.

값 설명 예제 코드
stretch 줄들을 교차축으로 가능한 범위까지 늘림. 
.container { align-content: stretch; }
flex-start 줄들을 교차축 시작점에 모음. 
.container { align-content: flex-start; }
flex-end 줄들을 교차축 끝점에 모음. 
.container { align-content: flex-end; }
center 줄들을 교차축 중앙에 모음. 
.container { align-content: center; }
space-between 첫 줄과 마지막 줄을 교차축 양 끝에 붙이고 줄 사이 간격 동일. 
.container { align-content: space-between; }
space-around 각 줄 주변 여백을 동일하게. 
.container { align-content: space-around; }
space-evenly 줄 사이와 양 끝 간격을 모두 동일하게. 
.container { align-content: space-evenly; }

추가 팁

  • 한 줄 레이아웃에서 align-content는 효과 없음: 반드시 flex-wrap이 wrap일 때만 의미가 있습니다.
  • justify-content vs align-items: 주축 정렬은 justify-content, 교차축 정렬은 align-items가 담당합니다.
  • flex-direction에 따라 축 변경: row/column에 따라 주축/교차축이 뒤바뀌니 정렬 속성의 체감 방향도 달라집니다.
  • 복합 사용: justify-content, align-items, flex-wrap, align-content를 함께 조합해 원하는 레이아웃을 만드세요.
  • 실무 조합 예시:
    • 헤더 메뉴 → flex-direction: row; justify-content: space-between;
    • 갤러리 → flex-wrap: wrap; align-content: center;
    • 버튼 그룹 → justify-content: center; align-items: baseline;
