CSS 3D Transform Z축 기초와 데모

전체 개요

CSS 3D Transform은 요소를 평면(2D)에서 벗어나 입체 공간으로 이동, 회전, 확대/축소할 수 있게 해줍니다.
X·Y·Z 세 축을 모두 다룰 수 있으며, perspective를 적용해 깊이감을 표현할 수 있습니다.
특히 Z축은 화면 깊이 방향을 담당하며, 양수는 사용자 쪽(앞으로), 음수는 화면 안쪽(뒤로)으로 이동합니다.

기초 설명

- translateZ(n): 요소를 Z축 방향으로 이동
- rotateZ(deg): 요소를 Z축을 기준으로 회전
- scaleZ(n): 요소를 Z축 방향으로 확대/축소

이 속성들은 큐브의 앞·뒤 면 배치, 회전, 두께감 표현에 직접적으로 관여합니다.
