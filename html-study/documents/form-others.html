Form 섹션 10 - 기타 유용한 입력 타입

전체 설명

HTML은 기본적인 텍스트 입력 외에도 다양한 특화 입력 타입을 제공합니다.
이들은 사용자가 더 직관적으로 데이터를 입력할 수 있도록 도와주며, 브라우저가 자동으로 적절한 UI를 제공해줍니다.
예를 들어 색상 선택기, 슬라이더, 검색창, URL·이메일 검증, 숫자·날짜·시간 입력 등이 있습니다.

속성 설명

• color: 색상 선택기를 제공해 사용자가 직접 색을 고를 수 있습니다.
• range: 슬라이더 형태로 숫자 범위를 선택할 수 있습니다.
• search: 검색 입력 전용 필드로, 일부 브라우저는 검색어 지우기 버튼을 제공합니다.
• url: URL 형식 검증을 지원해 올바른 주소만 입력되도록 합니다.
• email: 이메일 형식 검증을 지원해 올바른 이메일 주소만 입력되도록 합니다.
• number: 숫자 입력 전용 필드로, 최소/최대값과 step을 지정할 수 있습니다.
• date / time: 날짜와 시간을 선택할 수 있는 UI를 제공합니다.

속성별 정리

항목설명예제 코드데모
color 색상 선택기 
<input type="color">
range 슬라이더 
<input type="range" min="0" max="100" step="5">
search 검색 입력 
<input type="search" placeholder="검색어">
url URL 입력 (형식 검증) 
<input type="url" placeholder="https://example.com">
email 이메일 입력 (형식 검증) 
<input type="email" placeholder="you@example.com">
number 숫자 입력 
<input type="number" min="0" max="10" step="1">
date / time 날짜/시간 선택 
<input type="date"> <input type="time">
