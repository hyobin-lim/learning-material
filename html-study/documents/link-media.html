섹션2 - HTML 미디어 연결

문서 안에 시각·청각 자료를 포함해 풍부한 사용자 경험을 제공합니다. 이미지, 오디오, 비디오, 외부 임베드(YouTube, 지도 등)로 구분되며, 접근성을 위해 대체 텍스트, 자막, 설명 제공이 필수입니다.
실무에서는 반응형 이미지, 다양한 포맷 지원, 보안 속성 활용이 중요합니다.

미디어 활용 예시
요소 설명 예제 코드 데모
반응형 이미지 화면 크기에 따라 다른 이미지 제공 
<img src="small.jpg"
     srcset="large.jpg 800w, medium.jpg 400w"
     sizes="(max-width:600px) 400px, 800px"
     alt="반응형 이미지">
picture 요소 브라우저 지원에 따라 다른 포맷 제공 
<picture>
  <source srcset="image.webp" type="image/webp">
  <img src="image.jpg" alt="대체 이미지">
</picture>
오디오 기본 오디오 파일 재생 
<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>
오디오 autoplay/loop 자동 재생, 반복, 음소거 
<audio controls autoplay loop muted>
  <source src="song.mp3" type="audio/mpeg">
</audio>
비디오 (자막) 자막(track) 포함한 영상 
<video controls>
  <source src="movie.mp4" type="video/mp4">
  <track src="subtitles.vtt" kind="subtitles" srclang="ko" label="한국어">
</video>
비디오 poster/preload 썸네일 이미지와 로딩 방식 지정 
<video controls poster="thumbnail.jpg" preload="metadata">
  <source src="movie.mp4" type="video/mp4">
</video>
외부 비디오 (mp4) 외부 mp4 파일 삽입 
<video width="360" height="215" controls>
  <source src="mov-bbb.mp4" type="video/mp4">
</video>
YouTube 임베드 외부 플랫폼 영상 삽입 
<iframe width="360" height="215"
  src="https://www.youtube.com/embed/tgbNymZ7vqY"
  title="YouTube video"
  frameborder="0"
  allowfullscreen></iframe>
