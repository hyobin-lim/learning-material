CSS 박스 모델(Box Model) 완전 정리

1. 박스 모델이란?

모든 HTML 요소는 사각형 박스로 표현되며, 이 박스는 콘텐츠(Content), 패딩(Padding), 테두리(Border), 마진(Margin)으로 구성됩니다. 박스 모델은 요소의 크기와 화면 배치를 결정하는 핵심 개념입니다.

2. 구성 요소

구성설명관련 속성
Content 텍스트, 이미지 등 실제 내용이 들어가는 영역 width, height
Padding 콘텐츠와 테두리 사이의 안쪽 여백. 배경색/이미지가 패딩까지 확장됨 padding-top, padding-right, padding-bottom, padding-left
Border 콘텐츠와 패딩을 둘러싸는 테두리. 두께, 스타일, 색상 지정 가능 border-width, border-style, border-color
Margin 박스와 다른 요소 사이의 바깥 여백. 배경색은 적용되지 않음 margin-top, margin-right, margin-bottom, margin-left

3. 크기 계산

요소의 총 너비와 높이는 다음과 같이 계산됩니다: 

총 너비 = content width + padding 좌우 + border 좌우 + margin 좌우
총 높이 = content height + padding 상하 + border 상하 + margin 상하

5. Box-sizing

box-sizing 속성을 사용하면 크기 계산 방식을 바꿀 수 있습니다:

  • content-box (기본값): width/height는 콘텐츠 영역만 의미
  • border-box: width/height에 padding과 border까지 포함 → 레이아웃 관리가 쉬워짐

6. 핵심 요약

  • 박스 모델은 Content → Padding → Border → Margin 구조로 이루어짐
  • 총 크기는 콘텐츠 + 패딩 + 테두리 + 마진으로 계산
  • box-sizing으로 크기 계산 방식을 제어 가능
  • 모든 HTML 요소는 박스 모델을 기반으로 화면에 배치됨
