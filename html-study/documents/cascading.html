CSS 캐스캐이딩(Cascading) 기본

캐스캐이딩은 여러 CSS 규칙이 동일한 요소에 적용될 때, 어떤 규칙이 최종적으로 적용될지를 결정하는 과정입니다.

캐스캐이딩의 주요 원칙

원칙 설명 예시
출처(Source) 브라우저 기본 스타일, 사용자 스타일, 개발자가 작성한 스타일 중 우선순위 결정 사용자 정의 CSS가 브라우저 기본 스타일보다 우선
특이성(Specificity) 선택자의 구체성에 따라 우선순위가 달라짐 Inline > #id > .class > tag
중요도(Importance) !important가 붙은 규칙은 다른 규칙보다 우선 p { color:red !important; }
선언 순서(Order) 같은 우선순위일 경우, 나중에 선언된 규칙이 적용, 이전 것에 나중에 선언된 것이 덮여 쓰여짐 p { color:blue; }
p { color:green; }
→ 최종적으로 green 적용
