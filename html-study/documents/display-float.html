공통점과 차이점 정리

구분 display:inline-block float:left/right
공통점 요소들을 가로로 배치할 수 있으며, 블록 요소를 옆으로 나란히 배치하는 데 사용된다.
차이점 - 인라인 요소처럼 옆으로 배치되지만 블록 특성을 유지
- 부모 컨테이너의 흐름을 깨지 않음
- 세로 정렬(vertical-align) 속성 사용 가능 		- 요소를 왼쪽 또는 오른쪽으로 강제로 붙임
- 부모 컨테이너의 흐름을 깨뜨려 clearfix 필요
- 주로 레이아웃 구성에 사용
장점 간단히 옆으로 배치 가능, 레이아웃 안정적 복잡한 레이아웃 구현 가능, 이미지/텍스트 감싸기 용이
단점 인라인 특성 때문에 공백 문제 발생 가능 부모 높이 붕괴 문제 발생, clearfix 필요
사용 예시 버튼, 메뉴 아이템, 카드 레이아웃 신문 기사 레이아웃, 이미지와 텍스트 배치
