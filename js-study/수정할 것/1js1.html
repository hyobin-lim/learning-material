자바스크립트 기초 학습 문서

이 문서는 자바스크립트(JavaScript)의 기초 개념과 큰 틀을 설명하고, 주요 용어와 간단한 예제를 통해 직접 실행해볼 수 있도록 구성되어 있습니다.

1. 자바스크립트란?

  • 웹 브라우저에서 실행되는 프로그래밍 언어
  • HTML, CSS와 함께 웹의 3대 핵심 기술
  • 동적인 웹 페이지를 만들 수 있음 (버튼 클릭, 데이터 처리, 애니메이션 등)

2. 큰 틀에서의 구조

  • HTML: 구조 (뼈대)
  • CSS: 디자인 (색, 레이아웃)
  • JavaScript: 동작 (기능, 이벤트 처리)

3. 주요 용어

  • 변수 (Variable): 데이터를 저장하는 공간 (let, const)
  • 함수 (Function): 특정 작업을 묶어놓은 코드 블록 (function)
  • 이벤트 (Event): 사용자 행동 (클릭, 입력, 스크롤 등)
  • 객체 (Object): 관련 데이터와 기능을 묶은 구조 ({ })
  • DOM (Document Object Model): HTML 문서를 자바스크립트로 제어할 수 있게 만든 구조

4. 코드 예제와 데모

예제 1: 변수와 출력


// 예제 1: 변수
function demoVariable() {
  let name = "효빈";
  let age = 25;
  document.getElementById("demo1").textContent =
    "이름: " + name + ", 나이: " + age;
}

예제 2: 함수와 이벤트


// 예제 2: 함수
function demoFunction() {
  function greet(user) {
    return "안녕하세요, " + user + "님!";
  }
  document.getElementById("demo2").textContent = greet("효빈");
}

예제 3: DOM 제어

이 문장의 색상이 바뀝니다.


// 예제 3: DOM 제어
function changeColor() {
  const text = document.getElementById("demo3");
  text.style.color = text.style.color === "blue" ? "red" : "blue";
}

5. 정리

자바스크립트는 웹을 동적으로 만드는 핵심 언어입니다. 변수 → 함수 → 이벤트 → DOM 제어 순서로 기초 개념을 잡으면 전체적인 흐름을 이해하기 쉽습니다.
