자바스크립트 변수 배우기

변수는 데이터를 저장하고 재사용하기 위한 이름표입니다. 자바스크립트에서는 var, let, const 키워드를 사용하여 변수를 선언합니다.

🟢 변수란?

  • 정의: 데이터를 저장하는 이름표
  • 사용 이유: 반복되는 값을 효율적으로 관리하고, 코드의 가독성을 높임
  • 선언 키워드:
    • var : 오래된 방식, 함수 스코프
    • let : 블록 스코프, 값 변경 가능
    • const : 블록 스코프, 값 변경 불가(상수)

🟡 기본 예제와 설명

let name = "효빈";   // 문자열(String) 저장
const age = 25;      // 숫자(Number) 저장, 변경 불가
var city = "서울";   // 문자열(String) 저장, 오래된 방식

console.log(name);   // 출력: 효빈
console.log(age);    // 출력: 25
console.log(city);   // 출력: 서울

코드 구분 설명

  • let name = "효빈"; → name이라는 변수에 문자열 "효빈"을 저장
  • const age = 25; → age라는 변수에 숫자 25를 저장. const라서 값 변경 불가
  • var city = "서울"; → city라는 변수에 문자열 "서울"을 저장. (현대 코드에서는 잘 쓰지 않음)
  • console.log(...) → 변수에 저장된 값을 화면(콘솔)에 출력

🖥️ HTML 실행 예제

아래 코드는 브라우저 화면에 변수 값을 출력합니다.

let name = "효빈";   // 문자열 저장
const age = 25;      // 숫자 저장, 변경 불가
var city = "서울";   // 오래된 방식

document.write("이름: " + name + "<br>");
document.write("나이: " + age + "<br>");
document.write("도시: " + city + "<br>");

🔵 추가 팁

  • 변수 이름은 알파벳, 숫자, `_`, `$`만 사용 가능 (첫 글자는 숫자 불가)
  • 의미 있는 이름을 사용하면 코드 이해가 쉬워짐 (예: let score = 90;)
  • const를 기본으로 쓰고, 값이 변해야 할 때만 let을 쓰는 습관이 좋음
