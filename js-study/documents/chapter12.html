자바스크립트의 시작 — Chapter 12

이 문서는 자바스크립트의 제어문(Control Statements)을 설명합니다. 제어문은 프로그램의 흐름을 조건에 따라 제어하거나 반복을 수행할 때 사용됩니다.

1. if문

<script>
  var age = 20;

  if (age >= 18) {
    console.log("성인입니다.");
  } else {
    console.log("미성년자입니다.");
  }
</script>
💻 결과 화면 예시: 콘솔에 "성인입니다."가 출력됩니다.

2. else if문

<script>
  var score = 85;

  if (score >= 90) {
    console.log("A 학점");
  } else if (score >= 80) {
    console.log("B 학점");
  } else {
    console.log("C 학점");
  }
</script>
📊 결과 화면 예시: 콘솔에 "B 학점"이 출력됩니다.

3. switch문

<script>
  var fruit = "사과";

  switch (fruit) {
    case "사과":
      console.log("사과를 선택했습니다.");
      break; // break 없으면 다음 case까지 실행됨
    case "바나나":
      console.log("바나나를 선택했습니다.");
      break;
    default:
      console.log("알 수 없는 과일입니다.");
  }
</script>
💻 결과 화면 예시: 콘솔에 "사과를 선택했습니다."가 출력됩니다.

4. for문

<script>
  for (var i = 1; i <= 5; i++) { // 조건이 false가 되면 종료
    console.log(i);
  }
</script>
📊 결과 화면 예시: 콘솔에 "1 2 3 4 5"가 출력됩니다.

5. while문

<script>
  var n = 1;

  while (n <= 3) { // 조건이 false가 되면 종료
    console.log(n);
    n++;
  }
</script>
💻 결과 화면 예시: 콘솔에 "1 2 3"이 출력됩니다.

6. do...while문

<script>
  var m = 1;

  do {
    console.log(m);
    m++;
  } while (m <= 3); // 조건이 false가 되면 종료
</script>
📊 결과 화면 예시: 콘솔에 "1 2 3"이 출력됩니다.

7. break와 continue

<script>
  for (var i = 1; i <= 5; i++) {
    if (i === 3) break;      // 반복 종료
    console.log(i);
  }

  for (var j = 1; j <= 5; j++) {
    if (j === 3) continue;   // 3 건너뛰기
    console.log(j);
  }
</script>
💻 결과 화면 예시: 첫 번째 반복문은 "1 2" 출력, 두 번째 반복문은 "1 2 4 5" 출력됩니다.

8. 중첩 반복문

<script>
  for (var i = 1; i <= 2; i++) {
    for (var j = 1; j <= 3; j++) {
      console.log(i, j);
    }
  }
</script>
📊 결과 화면 예시: 콘솔에 "1 1", "1 2", "1 3", "2 1", "2 2", "2 3"이 출력됩니다.

9. 조건문과 논리 연산자 결합

<script>
  var isMember = true;
  var hasCoupon = false;

  if (isMember || hasCoupon) {
    console.log("할인 적용!");
  }
</script>
💻 결과 화면 예시: 콘솔에 "할인 적용!"이 출력됩니다.

10. 실습 과제

  • if문을 사용해 나이에 따라 성인/미성년자 출력하기
  • else if문으로 점수에 따라 학점 출력하기
  • switch문으로 과일 이름에 따라 메시지 출력하기
  • for문으로 1부터 10까지 출력하기
  • while문으로 1부터 5까지 출력하기
  • do...while문으로 1부터 3까지 출력하기
  • break와 continue를 활용해 반복문 제어하기
  • 중첩 반복문으로 구구단 출력하기
  • 조건문과 논리 연산자를 결합해 할인 여부 출력하기
