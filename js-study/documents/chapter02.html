자바스크립트의 시작 — Chapter 02

이 문서는 자바스크립트의 입출력(alert, confirm, prompt), document.write, console.log, 오류 확인 방법, 코딩 컨벤션을 설명합니다. 각 예제에는 실행 결과 설명을 포함했습니다.

1. 입출력하기

1.1 알림창 (alert)

<script>
  // alert() → 단순 알림 메시지를 띄움
  alert('안녕하세요?');
</script>
💻 실행 결과: 브라우저 화면에 "안녕하세요?"라는 팝업 알림창이 뜨고, 사용자가 확인 버튼을 눌러야 진행됩니다.

1.2 확인창 (confirm)

<script>
  // confirm() → 확인/취소 버튼이 있는 대화창을 띄움
  var reply = confirm('정말 배경 이미지를 바꾸겠습니까?');

  // 사용자의 선택을 콘솔에 출력합니다.
  console.log('사용자 선택:', reply);

  if (reply) {
    console.log('배경 이미지를 변경합니다.');
  } else {
    console.log('변경을 취소했습니다.');
  }
</script>
📊 실행 결과: 확인 버튼을 누르면 콘솔에 "true"와 "배경 이미지를 변경합니다."가 출력되고, 취소 버튼을 누르면 "false"와 "변경을 취소했습니다."가 출력됩니다.

1.3 프롬프트창 (prompt)

<script>
  // prompt() → 사용자에게 입력을 받을 수 있는 대화창을 띄움
  var name = prompt('이름을 입력하세요.', '아무개');

  if (name === null) {
    console.log('입력이 취소되었습니다.');
  } else {
    console.log('입력된 이름:', name);
    alert(name + '님, 환영합니다!');
  }
</script>
💻 실행 결과: 입력창이 뜨고 이름을 입력하면 콘솔에 "입력된 이름: OOO"가 출력되고, 알림창으로 "OOO님, 환영합니다!"가 표시됩니다. 취소하면 "입력이 취소되었습니다."가 출력됩니다.

1.4 document.write

<script>
  // document.write() → HTML 문서에 직접 내용을 출력
  document.write('<h1>안녕하세요</h1>');

  var name = prompt('이름을 입력하세요.');
  if (name) {
    document.write('<b><big>' + name + '</big></b>님, 환영합니다.');
  }
</script>
📊 실행 결과: 화면에 "안녕하세요"라는 큰 제목이 표시되고, 이름을 입력하면 입력한 이름과 함께 환영 메시지가 출력됩니다.

2. console.log

<script>
  var name = prompt('이름을 입력하세요.');
  console.log(name + '님, 환영합니다.');

  var age = 20;
  console.log('이름:', name, '나이:', age);
</script>
💻 실행 결과: 개발자 도구(Console)에 "OOO님, 환영합니다."와 "이름: OOO 나이: 20"이 출력됩니다.

3. 오류 확인

<script>
  var now = new Date();
  var display = now.toLocaleTimeString();

  // 오타: writ → 오류 발생
  document.writ('현재 시각은 ' + display);
</script>
⚠️ 실행 결과: 콘솔에 "Uncaught TypeError: document.writ is not a function" 오류가 표시됩니다.
<script>
  var now = new Date();
  var display = now.toLocaleTimeString();

  // 올바른 메서드명 write 사용
  document.write('현재 시각은 ' + display);
</script>
💻 실행 결과: 화면에 "현재 시각은 HH:MM:SS"가 표시됩니다.
<script>
  var name = "효빈";
  // 오타: nam → ReferenceError 발생
  console.log(nam);
</script>
⚠️ 실행 결과: 콘솔에 "Uncaught ReferenceError: nam is not defined" 오류가 표시됩니다.

4. 코딩 컨벤션

4.1 들여쓰기

포함 관계를 들여쓰기로 표현합니다. 보통 2칸 또는 4칸을 사용합니다.

var sum = n + 10; // 👍

*/

4.5 식별자 관례

  • PascalCase: UserCard
  • camelCase: checkTime()

  • kebab-case: click-handler
