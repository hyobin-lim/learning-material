자바스크립트의 시작 — Chapter 03

이 문서는 자바스크립트의 조건문과 반복문을 설명합니다. 프로그램의 흐름을 제어하는 방법을 배우며, 각 예제에는 실행 결과 설명을 포함했습니다.

1. 조건문

1.1 if문

<script>
  var score = prompt('점수를 입력하세요.'); // 사용자 입력을 문자열로 받음
  score = Number(score); // 문자열을 숫자로 변환

  if (score >= 60) {
    alert('합격입니다!');
  }
</script>
💻 실행 결과: 60점 이상 입력 시 "합격입니다!" 알림창이 표시됩니다. 60점 미만이면 아무 메시지도 표시되지 않습니다.

1.2 if-else문

<script>
  var score = prompt('점수를 입력하세요.');
  score = Number(score);

  if (score >= 60) {
    alert('합격입니다!');
  } else {
    alert('불합격입니다.');
  }
</script>
📊 실행 결과: 60점 이상이면 "합격입니다!", 그렇지 않으면 "불합격입니다."가 표시됩니다.

1.3 if-else if문

<script>
  var score = prompt('점수를 입력하세요.');
  score = Number(score);

  if (score >= 90) {
    alert('A 학점');
  } else if (score >= 80) {
    alert('B 학점');
  } else if (score >= 70) {
    alert('C 학점');
  } else if (score >= 60) {
    alert('D 학점');
  } else {
    alert('F 학점');
  }
</script>
💻 실행 결과: 입력한 점수에 따라 A~F 학점이 알림창으로 표시됩니다.

1.4 switch문

<script>
  var fruit = prompt('좋아하는 과일을 입력하세요.');

  switch(fruit) {
    case '사과':
      alert('사과를 선택했습니다.');
      break;
    case '바나나':
      alert('바나나를 선택했습니다.');
      break;
    case '포도':
      alert('포도를 선택했습니다.');
      break;
    default:
      alert('알 수 없는 과일입니다.');
  }
</script>
📊 실행 결과: 입력한 과일에 따라 해당 메시지가 표시됩니다. 다른 과일을 입력하면 "알 수 없는 과일입니다."가 표시됩니다.

2. 반복문

2.1 for문

<script>
  for (var i = 1; i <= 5; i++) {
    console.log(i + '번째 반복');
  }
</script>
💻 실행 결과: 콘솔에 "1번째 반복"부터 "5번째 반복"까지 출력됩니다.

2.2 while문

<script>
  var i = 1;
  while (i <= 5) {
    console.log(i + '번째 반복');
    i++;
  }
</script>
📊 실행 결과: 콘솔에 "1번째 반복"부터 "5번째 반복"까지 출력됩니다.

2.3 do-while문

<script>
  var i = 1;
  do {
    console.log(i + '번째 반복');
    i++;
  } while (i <= 5);
</script>
💻 실행 결과: 콘솔에 "1번째 반복"부터 "5번째 반복"까지 출력됩니다. 조건을 나중에 검사하기 때문에 최소 1번은 실행됩니다.

2.4 배열 반복

<script>
  var fruits = ['사과', '바나나', '포도'];
  for (var i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
  }
</script>
📊 실행 결과: 콘솔에 "사과", "바나나", "포도"가 순서대로 출력됩니다.

3. 반복 제어

<script>
  for (var i = 1; i <= 10; i++) {
    if (i === 5) {
      continue; // 5는 건너뜀 (반복은 계속 진행)
    }
    if (i === 8) {
      break; // 8에서 반복 종료 (전체 반복 종료)
    }
    console.log(i);
  }
</script>
⚙️ 실행 결과: 콘솔에 1~4, 6~7이 출력되고, 8에서 반복이 종료됩니다.

4. 실습 과제

  • prompt로 숫자를 입력받아 짝수/홀수 판별하기 (if문 활용)
  • prompt로 점수를 입력받아 학점(A~F) 출력하기 (if-else if문 활용)
  • for문으로 1부터 10까지 출력하기
  • while문으로 1부터 10까지 출력하기
  • for문에서 continue와 break를 활용해 특정 숫자를 건너뛰거나 종료하기
  • 배열에 저장된 과일 목록을 for문으로 출력하고, '바나나'만 건너뛰기
