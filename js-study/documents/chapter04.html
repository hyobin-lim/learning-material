자바스크립트의 시작 — Chapter 04

이 문서는 자바스크립트의 함수(Function)를 설명합니다. 함수는 코드의 재사용성을 높이고 프로그램을 구조적으로 작성할 수 있게 합니다.

1. 함수의 기본

<script>
  // 함수 선언
  function greet() {
    alert('안녕하세요!');
  }

  // 함수 호출
  greet();
</script>
💻 실행 결과 예시: "안녕하세요!"라는 알림창이 표시됩니다.

2. 매개변수와 인자

<script>
  // 매개변수(parameter)를 가진 함수
  function greet(name) {
    alert(name + '님, 환영합니다!');
  }

  // 인자(argument)를 전달하여 호출
  greet('효빈');
</script>
📊 실행 결과 예시: "효빈님, 환영합니다!"라는 알림창이 표시됩니다.

3. 반환값

<script>
  // 두 수를 더하는 함수
  function add(a, b) {
    return a + b;
  }

  var result = add(3, 5);
  console.log('결과:', result);
</script>
💻 실행 결과 예시: 콘솔에 "결과: 8"이 출력됩니다.

4. 함수 표현식

<script>
  // 함수 표현식 (익명 함수)
  var multiply = function(x, y) {
    return x * y;
  };

  console.log(multiply(4, 5));
</script>
📊 실행 결과 예시: 콘솔에 "20"이 출력됩니다.

5. 화살표 함수

<script>
  // 화살표 함수 → 간결한 문법, this 바인딩이 없음
  var divide = (x, y) => x / y;

  console.log(divide(10, 2));
</script>
💻 실행 결과 예시: 콘솔에 "5"가 출력됩니다.

6. 기본값 매개변수

<script>
  // 기본값 매개변수 (ES6)
  function greet(name = '손님') {
    alert(name + '님, 환영합니다!');
  }

  greet(); // 인자를 전달하지 않으면 '손님님, 환영합니다!' 출력
</script>
📊 실행 결과 예시: 알림창에 "손님님, 환영합니다!"가 표시됩니다.

7. 함수의 스코프

<script>
  function testScope() {
    var localVar = '지역 변수';
    console.log(localVar);
  }

  testScope();
  console.log(localVar); // 오류 발생: localVar is not defined
</script>
⚠️ 실행 결과 예시: 첫 번째 콘솔에는 "지역 변수"가 출력되고, 두 번째 콘솔에서는 ReferenceError가 발생합니다.

8. 함수의 활용

<script>
  // 조건문과 반복문을 함수와 결합
  function printEvenNumbers(limit) {
    for (var i = 1; i <= limit; i++) {
      if (i % 2 === 0) {
        console.log(i);
      }
    }
  }

  printEvenNumbers(10);
</script>
📊 실행 결과 예시: 콘솔에 2, 4, 6, 8, 10이 출력됩니다.

9. 실습 과제

  • 두 수를 입력받아 합을 반환하는 함수 작성하기
  • 이름을 입력받아 "OOO님, 환영합니다!"를 출력하는 함수 작성하기
  • 배열을 받아 모든 요소를 출력하는 함수 작성하기
  • for문과 함수를 결합해 1부터 100까지의 합을 구하는 함수 작성하기
  • 기본값 매개변수를 활용한 함수 작성하기
  • 스코프를 확인하는 함수 작성하기
