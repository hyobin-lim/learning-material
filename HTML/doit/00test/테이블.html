HTML Table 태그

1. 기본 구조

<table>: 표 전체를 감싸는 컨테이너

<tr>: 표의 행(row)

<td>: 표의 데이터 셀(cell)

<th>: 표의 헤더 셀

예시

<table>
  <tr>
    <th>이름</th>
    <th>나이</th>
  </tr>
  <tr>
    <td>김철수</td>
    <td>25</td>
  </tr>
</table>

결과

이름 나이
김철수 25

2. 구조 요소

<caption>

표의 제목을 정의합니다.

<table>
  <caption>학생 명단</caption>
  <tr>
    <th>이름</th>
  </tr>
</table>

<thead>, <tbody>, <tfoot>

표를 논리적으로 구분합니다.

  • <thead>: 머리글 영역
  • <tbody>: 본문 영역
  • <tfoot>: 바닥글 영역
<table>
  <thead>
    <tr>
      <th>항목</th>
      <th>금액</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>사과</td>
      <td>3000원</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>합계</td>
      <td>3000원</td>
    </tr>
  </tfoot>
</table>

결과

항목 금액
사과 3000원
합계 3000원

3. 셀 병합

colspan

열을 병합합니다.

<table>
  <tr>
    <th colspan="2">개인정보</th>
  </tr>
  <tr>
    <td>이름</td>
    <td>나이</td>
  </tr>
</table>

결과

개인정보
이름 나이

rowspan

행을 병합합니다.

<table>
  <tr>
    <th rowspan="2">학생</th>
    <td>국어</td>
  </tr>
  <tr>
    <td>수학</td>
  </tr>
</table>

결과

학생 국어
수학

4. 열 그룹

<colgroup>, <col>

열의 스타일을 지정합니다.

<table>
  <colgroup>
    <col style="background-color: #f0f0f0">
    <col>
  </colgroup>
  <tr>
    <td>A</td>
    <td>B</td>
  </tr>
</table>

5. 접근성 속성

scope

<th> 태그가 어떤 셀의 헤더인지 명시합니다.

  • row: 행 헤더
  • col: 열 헤더
<table>
  <tr>
    <th scope="col">이름</th>
    <th scope="col">나이</th>
  </tr>
  <tr>
    <th scope="row">학생1</th>
    <td>20</td>
  </tr>
</table>

headers

복잡한 표에서 셀과 헤더를 연결합니다.

<table>
  <tr>
    <th id="n">이름</th>
    <th id="a">나이</th>
  </tr>
  <tr>
    <td headers="n">김철수</td>
    <td headers="a">25</td>
  </tr>
</table>
