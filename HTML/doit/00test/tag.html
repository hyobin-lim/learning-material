HTML 부모-자식-손자 태그 상세 구조

아래 표는 HTML에서 부모 태그와 그 자식, 손자 태그를 더 다양하고 상세히 정리한 것입니다.

문서 기본 구조
부모 태그 자식 태그 손자 태그 예시 설명
<html> <head>, <body> <head> → <title>, <meta>, <link>, <style>, <script>
<body> → <header>, <nav>, <main>, <section>, <footer> 
<html>
  <head>
    <title>문서 제목</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="style.css">
    <script src="app.js"></script>
  </head>
  <body>
    <header><h1>머리말</h1></header>
    <main><section>본문</section></main>
  </body>
</html>
 문서 전체를 감싸는 루트 요소.
<header> <h1>–<h6>, <nav>, <p> <nav> → <ul> → <li> → <a> 
<header>
  <h1>사이트 제목</h1>
  <nav>
    <ul>
      <li><a href="#">홈</a></li>
      <li><a href="#">소개</a></li>
    </ul>
  </nav>
</header>
 머리말에는 제목, 내비게이션, 소개 문구 등을 포함.
<section> <h2>, <p>, <article>, <aside> <article> → <h3>, <p>, <figure> 
<section>
  <h2>섹션 제목</h2>
  <article>
    <h3>기사 제목</h3>
    <p>본문 내용</p>
    <figure>
      <img src="img.jpg" alt="이미지">
      <figcaption>이미지 설명</figcaption>
    </figure>
  </article>
</section>
 섹션은 기사, 본문, 이미지 등을 포함 가능.
<table> <caption>, <thead>, <tbody>, <tfoot> <thead> → <tr> → <th>
<tbody> → <tr> → <td> → <p>, <a>, <img> 
<table>
  <caption>회원 목록</caption>
  <thead>
    <tr><th>이름</th><th>나이</th></tr>
  </thead>
  <tbody>
    <tr>
      <td><p>홍길동</p></td>
      <td><img src="icon.png" alt="아이콘"></td>
    </tr>
  </tbody>
</table>
 표의 셀 안에는 문단, 이미지, 링크 등 다양한 콘텐츠 가능.
<form> <fieldset>, <label>, <input>, <textarea>, <select>, <button> <fieldset> → <legend>, <input>
<select> → <optgroup> → <option> 
<form>
  <fieldset>
    <legend>회원 가입</legend>
    <label>아이디: <input type="text"></label>
    <label>소개: <textarea></textarea></label>
    <select>
      <optgroup label="지역">
        <option>서울</option>
        <option>부산</option>
      </optgroup>
    </select>
    <button type="submit">제출</button>
  </fieldset>
</form>
 폼은 입력 요소, 선택 항목, 그룹화를 포함.
<figure> <img>, <figcaption> <figcaption> → 텍스트, 인라인 태그(<strong>, <em>) 
<figure>
  <img src="photo.jpg" alt="사진">
  <figcaption><strong>중요</strong> 사진 설명</figcaption>
</figure>
 이미지와 설명을 함께 표시, 설명 안에 강조 태그 가능.
<details> <summary>, 기타 콘텐츠 <summary> → 텍스트, 인라인 태그 
<details>
  <summary><em>더 보기</em></summary>
  숨겨진 내용 <p>추가 설명</p>
</details>
 펼침/접기 가능한 영역, 요약 안에 인라인 태그 포함 가능.
