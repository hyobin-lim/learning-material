img 태그의 alt 속성 완벽 가이드

초보자를 위한 alt 속성 사용법과 예제

1. alt 속성이란?

alt = "Alternative Text" (대체 텍스트)

이미지를 설명하는 텍스트입니다. 이미지가 표시되지 않을 때 대신 나타나는 글자입니다.

<img> 태그에서 alt 속성은 이미지에 대한 설명을 담고 있으며, 다음과 같은 역할을 합니다:

  • 이미지 로드 실패 시 그 자리에 표시되는 텍스트
  • 시각장애인을 위한 스크린 리더기가 읽어주는 텍스트
  • 검색 엔진(Google, Naver 등)이 이미지 내용을 이해하는 데 도움
  • 웹 접근성(Accessibility) 개선

2. alt 속성 사용 규칙

상황 alt 값 설명 예시
의미 있는 이미지
(상품, 뉴스 사진 등)		 이미지 설명 입력 이미지가 의미를 가질 때 반드시 입력 alt="레드향 과일 사진"
장식용 이미지
(배경, 구분선 등)		 빈 값
alt=""		 그냥 꾸미는 용도라면 비워두기 alt=""
로고 회사/서비스명 브랜드 이름 입력 alt="네이버 로고"
아이콘 용도 설명 아이콘이 나타내는 기능 설명 alt="검색 버튼"
그래프, 차트 데이터 요약 그래프가 보여주는 주요 정보 alt="2025년 판매량 증가 그래프"

3. 코드 예제

❌ 잘못된 사용

<!-- alt가 비어있음 -->
<img src="images/tangerines.jpg" alt="">

<!-- alt 값이 너무 길거나 불필요함 -->
<img src="logo.png" alt="이것은 로고입니다 로고입니다 로고입니다">

<!-- alt가 완전히 없음 -->
<img src="product.jpg">

✅ 올바른 사용

<!-- 상품 이미지: 짧고 명확하게 -->
<img src="images/tangerines.jpg" alt="레드향">

<!-- 로고: 회사명 입력 -->
<img src="logo.png" alt="네이버 로고">

<!-- 장식용: alt 비워두기 -->
<img src="decoration.jpg" alt="">

<!-- 상세한 설명 (필요시) -->
<img src="chart.png" alt="2025년 Q1~Q4 월별 판매량 증가 추이">

현재 파일 수정 예제

수정 전:

<img src="images/tangerines.jpg" alt="">

수정 후:

<img src="images/tangerines.jpg" alt="레드향">

4. alt 속성 작성 팁

  1. 짧고 명확하게: 10~15자 정도가 적당합니다.
  2. "이미지입니다" 같은 설명 피하기: 이미지라는 것은 이미 알려져 있습니다.
  3. 마침표 사용 최소화: 스크린 리더가 연속으로 읽을 때 자연스럽게 합니다.
  4. 중요한 정보 우선: 사용자가 가장 알고 싶은 정보를 먼저 작성합니다.
  5. 장식용은 alt="": 의미 없는 이미지는 반드시 비워야 합니다.

5. 실제 예제 비교

상황 좋은 alt 나쁜 alt
상품 사진 alt="레드향" alt="빨간 과일"
사람 프로필 사진 alt="김철수 CEO 프로필" alt="사람 사진"
검색 아이콘 alt="검색" alt="검색 아이콘입니다"
배경 이미지 alt="" alt="배경"

6. 왜 alt 속성이 중요한가?

  • 접근성: 시각장애인도 웹 사이트를 이용할 수 있습니다.
  • SEO: 검색 엔진이 이미지를 이해하므로 검색 결과에 잘 나타납니다.
  • 사용자 경험: 이미지 로드 실패 시 사용자가 무엇인지 알 수 있습니다.
  • 웹 표준: HTML 표준에서 권장하는 좋은 습관입니다.
  • 법적 의무: 많은 국가에서 접근성 관련 법규를 강화하고 있습니다.
