heading 태그란?

heading(h1~h6)은 문서의 제목을 나타내는 태그입니다. 숫자가 작을수록 더 중요한 제목입니다.

예시 소스코드

아래는 HTML 코드가 그대로 보이도록 한 예시입니다.

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>예시 페이지</title>
</head>
<body>
  <h1>페이지 제목 (h1)</h1>
  <h2>섹션 제목 (h2)</h2>
  <p>본문 예시...</p>
</body>
</html>

간단한 사용 팁

  • 논리적 계층(h1>h2>h3...)을 지키세요.
  • 보이는 크기는 CSS로 조절하고 의미는 태그로 남겨두세요.
  • 접근성과 SEO에 도움이 됩니다.
