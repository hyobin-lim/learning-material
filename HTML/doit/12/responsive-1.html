CSS Grid 주요 템플릿 속성 설명

설명 코드 데모
grid-template-rows
그리드의 행(row) 높이를 정의합니다. 
.container {
  display: grid;
  grid-template-rows: 50px 100px 1fr;
}

grid-template-columns
그리드의 열(column) 너비를 정의합니다. 
.container {
  display: grid;
  grid-template-columns: 100px 1fr 2fr;
}
1
2
3
grid-template-areas
이름이 지정된 영역을 정의하여 요소를 배치합니다. 
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 100px 1fr;
  grid-template-rows: 50px 1fr 30px;
}

grid-auto-rows
명시적으로 정의되지 않은 행의 기본 높이를 설정합니다. 
.container {
  display: grid;
  grid-template-columns: 100px;
  grid-auto-rows: 60px;
}

grid-auto-columns
명시적으로 정의되지 않은 열의 기본 너비를 설정합니다. 
.container {
  display: grid;
  grid-template-rows: 60px;
  grid-auto-columns: 80px;
}

grid-auto-flow
자동 배치 방향을 설정합니다. 기본값은 row이며, column으로 설정하면 열 방향으로 배치됩니다. 
.container {
  display: grid;
  grid-template-columns: repeat(2, 80px);
  grid-auto-flow: column;
}
