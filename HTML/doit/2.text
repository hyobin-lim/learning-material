## 2. HTML 기본 구조와 문법
- **소제목:** HTML 기본구조

  - 실습: 기본 문서 작성하기  
- **소제목:** HTML 기본 문법  
  - 내용: 태그, 속성, 계층 구조  
  - 실습: 간단한 페이지 작성  
- **소제목:** Entity 코드표  
  - 내용: 특수문자 표현 (`&nbsp;`, `&copy;`)  
  - 실습: 텍스트에 특수문자 삽입  
- **소제목:** 텍스트 태그 정리

- **소제목:** 제목 태그  
  - 내용: `<h1>`~`<h6>`, `<blockquote>`, `<cite>`  
  - 실습: 기사 페이지 작성  
- **소제목:** 목록 태그  
  - 내용: 순서 없는 목록, 순서 있는 목록  
  - 실습: 쇼핑 리스트 만들기  
- **소제목:** 블록 레벨과 인라인 레벨 태그  
  - 내용: 태그 분류, 혼용 가능한 태그 비교

## 8. CSS 박스 모델과 레이아웃
- **소제목:** 박스 요소  
  - 내용: width, height, padding, margin  
  - 실습: 박스 모델 시각화  
- **소제목:** 마진과 패딩  
  - 내용: 차이점, 수평 중앙정렬

- **소제목:** Box-sizing  
  - 내용: `content-box`, `border-box`  
  - 실습: 크기 계산 비교  
- **소제목:** 레이아웃 방식 비교  
  - 내용: float, inline-block, flexbox, grid  
  - 실습: 동일 레이아웃을 여러 방식으로 구현  
- **소제목:** Box-sizing 동적 데모  
  - 내용: 실시간 크기 변화 확인  
  - 실습: 브라우저에서 실험

## 9. CSS 정렬과 배치
- **소제목:** CSS 정렬 속성  
  - 내용: `vertical-align`, `align-content`, `align-items`  
  - 실습: 텍스트와 박스 정렬 실험  
- **소제목:** margin을 활용한 중앙 정렬  
  - 내용: `margin: auto`를 이용한 블록 요소 중앙 배치  
  - 실습: 카드 박스 중앙 배치하기  
- **소제목:** float 속성  
  - 내용: `float: left/right`, `clear`  
  - 실습: 이미지와 텍스트 배치  
- **소제목:** float 해제 방법  
  - 내용: `clear`, `overflow`, `::after` 활용 등 5가지 방법  
  - 실습: 레이아웃 깨짐 해결하기  
- **소제목:** display 속성  
  - 내용: `block`, `inline`, `inline-block`, `none`  
  - 실습: 요소 표시 방식 변경  
- **소제목:** display vs float 비교  
  - 내용: `display:inline-block`과 `float:left/right` 차이  
  - 실습: 동일 레이아웃을 두 방식으로 구현
