break 문

    반복문을 실행하다가 break를 만나면
    그 즉시 반복을 멈추고 밖으로 나갑니다.

    (예시)
    for (let i = 1; i <= 5; i++) {
      if (i === 3) {
        break; // 3이 되면 탈출!
      }
      document.write(i);
    }

    결과: 1 2 (3부터는 실행 안 됨)

continue 문

    반복문을 실행하다가 continue를 만나면
    그 반복만 건너뛰고 다음 반복을 계속합니다.

    (예시)
    for (let i = 1; i <= 5; i++) {
      if (i === 3) {
        continue; // 3은 건너뛰고 계속!
      }
      document.write(i);
    }

    결과: 1 2 4 5 (3만 실행 안 됨)
